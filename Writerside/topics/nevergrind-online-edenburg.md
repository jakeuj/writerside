# Nevergrind Online Edenburg 伊登堡城鎮設施總覽

`Edenburg`（伊登堡）是 Nevergrind Online 每場冒險之間的整備中心：玩家在這裡接任務、補給、升技能、鑑定裝備、擴背包、管理銀行、處理 guild 與社交互動。把 Edenburg 的七大建築用熟，會比只知道「回城賣裝」更能穩定推進角色成長。

- 相關速查：[熱鍵與聊天指令速查](nevergrind-online-hotkeys-commands.md)
- 資料來源：來源摘要、來源資料整理與既有 Nevergrind Online 城鎮設施筆記
- 版本提醒：Edenburg 的 UI 位置、建築名稱、賽季規則、背包上限、guild 功能與快捷操作可能因版本或語系調整；實際操作請以目前遊戲內 UI、`/help` 與 tooltip 為準

<p>新手最常跑的循環是 Tavern 接任務，Apothecary 補給 / 鑑定，Merchant 賣裝 / 擴背包，Academy 升技能。</p>

## 七大建築總表

| 建築 | 核心功能 | 什麼時候去 |
| ------ | ------ | ------ |
| [Tavern 酒館](nevergrind-online-tavern.md) | `Missions`、地城入口、leaderboards、tips | 出發前選任務、看掉落偏好、查 tips、確認隊伍 ready |
| `Academy` 學院 | 學習 / 升級技能、重置 talents | 核心 rotation 不順、拿到新技能、想調整 build 時 |
| [Apothecary 藥劑店](nevergrind-online-apothecary.md) | 藥水、`Identify Scrolls`、`Identify All`、cloth / focus / stave | 出發前補給，回城後批量鑑定，caster / healer 補裝 |
| [Merchant 商店](nevergrind-online-merchant.md) | 背包擴張、leather armor、一般商品、gambling | 滿包、清戰利品、補輕型裝、後期用 gold 追特定部位 |
| [Blacksmith 鍛造鋪](nevergrind-online-blacksmith.md) | plate / mail、重型武器、item upgrades、runes / crafting | 重裝職業補裝，或準備把 rune 投進長期裝備 |
| `Bank` 銀行 | 帳號共享倉庫 | 存 set、unique、分身裝、抗性裝、高價交易品 |
| `Guild Hall` 公會大廳 | 建立 / 管理公會與 guild 狀態 | 固定團、社群聊天、長期玩家關係 |

這些建築的價值不是分散的。Edenburg 真正的節奏是：從 Tavern 選目標，進地城拿掉落，回城到 Apothecary 鑑定，經 Merchant / Bank 分流，再用 Academy、Blacksmith 或 gambling 把角色往下一階推。

## Tavern：冒險與資訊起點

`Tavern` 是把玩家送進內容的入口。來源摘要提到玩家需要在酒館選擇 `Missions` 才能前往隨機生成的地城；任務數量被整理為超過 100 種，但實際數量與分類仍以目前版本為準。

Tavern 的重點：

- `Missions`：選地城、任務目標與 farm 方向。
- 掉落偏好：來源摘要提到每項任務可能有偏好的裝備類型，適合用來規劃 farm。
- `Leaderboards`：查看等級、ladder 或賽季競爭狀態。
- `Tips`：新手優先看的知識入口，包含快捷鍵、基礎教學與聊天 / 組隊提示。

完整流程可看 [Tavern 酒館指南](nevergrind-online-tavern.md)。

<note>
<p>來源摘要提到 ladder / season 可能約三個月一輪。這類營運週期很容易調整，正式衝榜前請先看遊戲內公告與 leaderboard 說明。</p>
</note>

## Academy：技能成長與重置

`Academy` 是角色能力成長的訓練節點。當你發現技能傷害不夠、治療跟不上、rotation 太卡，或新技能已經解鎖，通常就該回來檢查技能等級與 talents。

Academy 常見用途：

- 消耗 gold 學習新技能。
- 升級現有技能，讓核心 rotation 成形。
- 付費重置 talents，重新調整 build。
- 在換裝、轉流派或組隊定位改變後重新分配點數。

前期 gold 常在技能、背包、補給之間拉扯。比較穩的順序是先把常用技能升到能順利打地城，再逐步投資背包、鑑定、裝備升級與 gambling。

## Apothecary：補給、鑑定與法系整備

`Apothecary` / `Alchemist` 是所有職業都會用到的後勤站，也是 caster / healer 常看的裝備商店。

它負責：

- 販售藥水，尤其是 `Spirit Potions` 類資源補給。
- 販售 `Identify Scrolls`，讓玩家在地城中即時判斷戰利品。
- 提供 `Identify All` 類批量鑑定服務。
- 販售 cloth armor 與 focus / stave 類法系鈍器。

對 healer 來說，spirit 補給直接關係到能不能撐完整場；對刷寶玩家來說，`Identify All` 則是把未知掉落轉成金幣或裝備判斷的捷徑。完整細節可看 [Apothecary 藥劑店指南](nevergrind-online-apothecary.md) 與 [Identification 鑑定指南](nevergrind-online-identification.md)。

## Merchant：背包、買賣與 gambling

`Merchant` 是 Edenburg 的資源管理槓桿。來源摘要把 `Twiddles` 與背包擴張放在這裡，並整理出背包可從初始 8 格逐步升到 40 格；這是角色獨立投資，每個角色都要自己付費。

Merchant 的核心價值：

- 擴張 inventory，讓每趟地城能帶回更多戰利品與補給。
- 販售 leather armor、一般商品與隨機裝備。
- 清理低價值戰利品，把掉落轉成 gold。
- 後期使用 [Gambling 賭博](nevergrind-online-gambling.md) 追 unique、set、legendary 或特定部位。

對剛開始 farming 的角色來說，背包擴張往往比多買一件過渡裝更有長期收益。完整策略可看 [Merchant 商店指南](nevergrind-online-merchant.md)。

## Blacksmith：重裝武庫與後期強化

`Blacksmith` 是重裝職業的武庫，也是 socketed items 後期客製化的重要節點。

它負責：

- 販售 plate armor、mail armor 與重型武器。
- 處理 item upgrades、rune / crafting 類後期強化。
- 讓 socketed items 透過 runes 補強 build 需要的屬性。

來源摘要提醒，rune 鑲嵌通常要慎重看待，因為投入後可能無法輕易取回。不要把高價 rune 塞進很快會換掉的裝備；先確認 roll、sockets、build 契合度與使用壽命。完整流程可看 [Blacksmith 鍛造鋪指南](nevergrind-online-blacksmith.md) 與 [符文 Runes 指南](nevergrind-online-runes.md)。

## Bank：帳號共享財產庫

`Bank` 是多角色發展的生命線。來源摘要把銀行空間整理為同帳號角色共享，這讓高等角色取得的裝備可以留給分身、交易或未來 build。

適合放進 Bank 的東西：

- set items 與缺件套裝。
- 高 roll unique、legendary 或值得交易的 rare。
- 抗性裝、特定族類特攻裝、資源恢復裝。
- 分身角色未來能穿的過渡裝。
- 高價 rune、socketed base 或暫時不想誤賣的物品。

Bank 的重點不是把所有東西都塞進去，而是保留「未來有明確用途」的裝備。若一件物品既不自用、不給分身、不交易，也沒有特殊 roll，通常可以賣掉換成下一輪 farm 資金。

## Guild Hall：社交連結與固定團

`Guild Hall` 用於建立、管理或確認 guild 狀態。來源摘要提到，目前加入公會不一定有直接屬性獎勵；它更像社交與固定團協作的入口，提供 guild channel、玩家關係與長期組隊的便利。

Guild Hall 的價值通常體現在：

- 固定隊伍與 farm 夥伴。
- guild chat 或專屬社群頻道。
- 交易、分裝、補位與新手協助。
- ladder / season 期間的協作與資訊交換。

如果你主要 solo，Guild Hall 可能不是每天都會用；但如果想長期刷高難度、追 ladder 或固定 farm，社群會比單純面板更重要。

## 城鎮社交與快捷操作

Edenburg 也承擔社交大廳的角色。來源摘要把它的聊天與好友系統描述成有經典 Battle.net 風格：玩家可以私聊、使用好友列表、在頻道中招募隊伍，或透過 UI 操作邀請其他玩家。

常用操作：

- `/say`：在城鎮或當前頻道發言，實際範圍以目前聊天 UI 為準。
- `/tell playerName message`：發送私訊。
- `/friend add playerName`：新增好友。
- `/join channelName`：加入指定頻道，例如 LFG 類頻道。
- `/camp` 或左上角 `Camp` 類按鈕：快速回到角色選擇畫面。
- 右鍵點擊玩家頭像或名稱：若目前 UI 支援，可進行邀請、私訊或其他互動。

更多按鍵與指令可以看 [熱鍵與聊天指令速查](nevergrind-online-hotkeys-commands.md)。如果舊攻略和目前 UI 不一致，以遊戲內 `/help` 與 `Esc` > `Hotkeys` 為準。

## 每趟地城後的整理順序

一趟地城結束後，可以照這個順序整理：

1. 到 Apothecary 補藥水，並用 `Identify All` 處理未鑑定裝備。
2. 先挑出自用、分身、交易或高 roll 候選，放到 Bank 或固定背包區。
3. 到 Merchant 賣掉低價戰利品，確認是否該升背包。
4. 若有 socketed item、重甲或重武器候選，再到 Blacksmith 評估升級 / rune plan。
5. 若核心技能跟不上目前難度，到 Academy 升技能或重配 talents。
6. 回到 Tavern 選下一個任務與 farm 目標。

這就是 Edenburg 的核心功能：它把每場地城的隨機掉落，轉成下一場更穩的技能、裝備、背包、資源與隊伍準備。
