## 地城探險與移動

| 操作 | 預設按鍵 / 動作 | 用途 |
| ------ | ------ | ------ |
| 基本移動 | `WASD` | 控制角色在地城中前後左右移動 |
| 自動行走 | `Z` | 切換 autorun；可到 `Esc` > `Hotkeys` 修改 |
| 小地圖移動 | 點擊 minimap 上的下一個房間 | 讓角色自動朝該方向前進 |
| 滑鼠前進 | 長按 `Left Click` 或 `Right Click` | 在地城中向前移動 |
| 持續前進技巧 | 按住左鍵移動時，把滑鼠拖到角色頭像區域後放開 | 可讓角色維持前進，適合長走廊或回頭清圖 |
| 地圖縮放 | `Mouse Wheel` | 在地城中縮放地圖，方便找隱藏房或確認路線 |

<note>
<p>如果 <code>Z</code> 或滑鼠移動手感不合，可以先到 <code>Esc</code> &gt; <code>Hotkeys</code> 檢查目前 keybind。不要只照舊攻略硬背，因為玩家自訂或版本更新都可能改掉預設值。</p>
</note>

## 戰鬥熱鍵

| 操作 | 預設按鍵 / 動作 | 用途 |
| ------ | ------ | ------ |
| 主要技能列 | `1` 到 `9`、`0`、`-`、`=` | 對應 12 個職業技能 |
| 第二組快捷技能 | `Shift + 1` 到 `Shift + 6` | 部分職業可用，例如遊俠（Ranger）、盜賊（Rogue）、暗影騎士（Shadow Knight）、薩滿（Shaman）、天騎士（Templar）、術士（Warlock）、戰士（Warrior） |
| 停止施法 | `Spacebar` | 立刻停止目前施法，用於危險轉向、換目標或避免放錯技能 |
| 交換技能位置 | 右鍵點一個技能圖示，等它閃爍後再右鍵點另一個圖示 | 交換快捷欄位置，讓常用循環更順手 |
| 選取隊友 | `F1` 到 `F5` | `F1` 選自己，`F2` 到 `F5` 對應隊伍中的其他玩家 |

`Spacebar` 很值得練成肌肉記憶。NGO 的戰鬥常需要臨時停手、切目標、等 tank 建立 aggro，或在 boss traits 讓施法變危險時立刻收招。

## 系統與介面

| 操作 | 預設按鍵 / 動作 | 用途 |
| ------ | ------ | ------ |
| 角色視窗 | `C` 或 `H` | 查看裝備、詳細數值、HP / mana / spirit 與經驗進度 |
| 回覆私訊 | `R` | 快速回覆上一個 tell / whisper 對象 |
| 重叫聊天訊息 | `Up Arrow` | 找回先前送出的訊息，常用於 LFG / LFM 或重複招募 |
| 主選單 | `Esc` | 開啟選單，包含 `Hotkeys` 設定 |

如果你剛進遊戲不知道某個系統在哪裡，先試 `Esc`、酒館 `Tips` 分頁與 `/help`。這三個入口通常比翻舊截圖更可靠。

## 物品、商店與銀行

| 操作 | 預設按鍵 / 動作 | 用途 |
| ------ | ------ | ------ |
| 貼物品到聊天 | `Shift + Left Click` | 把物品資訊貼到聊天頻道，方便交易、詢價或討論 |
| 比較裝備 | 滑鼠移到物品上時按住 `Shift` | 與目前已裝備的同部位物品比較 |
| 商店快速買賣 | 商店視窗開啟時 `Ctrl + Left Click` | 快速買入或賣出物品 |
| 銀行快速取出 | 銀行開啟時 `Shift + Right Click` | 將銀行物品快速移入背包 |
| 銀行 / 背包移動 | 銀行開啟時 `Right Click` | 常用於銀行與背包間移動物品；實際方向與支援程度看目前版本 UI |

物品操作最怕誤賣或誤移。高價 unique、set、legendary、socketed item 或帶 talents 的候選裝，建議先放到固定背包區或銀行，再開始批量賣垃圾。

## 常用聊天指令 {#ngo-chat-command-quick-reference}

| 指令 | 用途 |
| ------ | ------ |
| `/help` | 查看目前版本可用的完整 help / command 清單 |
| `/help party` | 查看組隊相關 help；若目前版本支援，這通常是查 party 指令最快的入口 |
| `/who` | 查詢線上玩家 |
| `/who 5 10 dwarf cleric` | 依等級、種族、職業等條件過濾線上玩家 |
| `/friend add playerName` | 新增好友 |
| `/tell playerName message` | 發送私訊 |
| `/join channelName` | 加入指定聊天頻道，例如 LFG 類頻道 |
| `/camp` | 回到角色選擇畫面 |
| `/played` | 顯示角色建立時間與總遊玩時數 |

<warning>
<p>舊攻略可能會列出一些現在不一定存在的指令。只要和目前遊戲內 <code>/help</code> 不一致，先以遊戲內 help、酒館 tips 或官方公告為準。</p>
</warning>

## 新手先練哪幾個

1. 先把 `WASD`、`Z`、小地圖點房間與滑鼠滾輪縮放練熟。
2. 把主循環技能放到 `1` 到 `=` 裡最順手的位置。
3. 練會用 `Spacebar` 停止施法，避免危險時技能卡住。
4. 回城後用 `Shift` 比較裝備，再用 `Ctrl + Left Click` 清理商店買賣。
5. 組隊或交易前先熟悉 `/who`、`/tell`、`/join`、`R` 與 `Up Arrow`。

操作熟了以後，可以接著看 [酒館（Tavern）指南](nevergrind-online-tavern.md) 了解出發與 ready 流程，或看 [商人（Merchant）指南](nevergrind-online-merchant.md) 整理背包、買賣與 gambling 的後勤節奏。
