# Nevergrind Online Apothecary 藥劑店指南

Nevergrind Online 的 `Apothecary` 藥劑店，也可能在部分資料中被稱為 `Alchemist`，是 Edenburg 裡支撐冒險續航的補給站，也是 caster / healer 常回頭檢查裝備的法系商店。出發前買好藥水與 `Identify Scrolls`，回城後用 `Identify All` 類服務處理戰利品，再檢查 cloth armor、focus 或 stave 是否剛好補到 talents / 資源缺口，會讓每一趟地城更順。

- 資料來源：NotebookLM 摘要、Fandom / 公開 wiki 與來源資料整理
- 版本提醒：Apothecary / Alchemist 名稱、位置、補給品項、`Identify All` 按鈕、restock 週期與裝備品類可能因版本或語系調整；實際操作請以目前遊戲內 UI 與 tooltip 為準

<tldr>
<p>Apothecary 是地城前後的補給與鑑定中心，負責藥水、鑑定卷軸與 <code>Identify All</code> 類服務。</p>
<p><code>Spirit Potions</code> 對 healer / buffer 尤其重要，因為 spirit 類資源常是續航瓶頸。</p>
<p>它也是 caster 向商店，會販售 cloth armor、focus / stave 類法系武器與可能帶 talents 的隨機裝備。</p>
</tldr>

## Apothecary 在 Edenburg 的定位

Edenburg 的商業設施有明確分工。`Blacksmith` 供應重甲與重武器，`Merchant` 以背包、皮甲與一般商品為主，`Apothecary` 則負責補給、鑑定與法系裝備。來源摘要把它描述為城鎮中心偏左的設施；如果版本或語系不同，請以目前城鎮 UI 為準。

| 設施 | 裝備 / 功能重點 | 使用情境 |
| ------ | ------ | ------ |
| `Apothecary` / `Alchemist` | 藥水、`Identify Scrolls`、`Identify All`、cloth armor、focus / stave | 地城前補給、回城後批量鑑定、caster / healer 補裝 |
| `Blacksmith` | plate、mail、重型武器、runes / crafting | 重裝職業補裝、後期 socketed items 客製化 |
| `Merchant` | 背包擴張、leather armor、一般商品、隨機庫存 | 提升刷寶續航、賣低價戰利品、檢查輕型裝備 |

對所有職業來說，Apothecary 是把地城掉落轉成可判斷裝備的入口；對 `Cleric`、`Druid`、`Shaman`、`Wizard`、`Enchanter` 等 caster / healer 來說，它還是補資源與找法系過渡裝的重要節點。

## 藥水與 Spirit 續航

來源摘要特別強調 `Spirit Potions` 的價值：`Spirit` 類資源恢復較慢，而且許多治療、buff 或支援技能都會吃到這條資源。這讓 Apothecary 對 healer / support 來說不只是便利商店，而是能不能撐完高壓地城的生命線。

出發前建議檢查：

1. 目前地城長度與難度，是否需要多帶 spirit / mana / health 類補給。
2. healer 是否需要額外 `Spirit Potions` 來支撐補血與 buff uptime。
3. caster 是否常因 mana / spirit 不足而斷 rotation。
4. 背包空間是否足夠；若不夠，先看 [Merchant 商店指南](nevergrind-online-merchant.md) 的背包擴張策略。
5. 是否要帶 `Identify Scrolls`，在地城中即時篩掉低價掉落。

<tip>
<p>如果隊伍在 boss 戰後半段常常不是死於傷害，而是 healer 資源乾掉，先補 spirit / mana 續航通常比多一點帳面輸出更有用。</p>
</tip>

## 鑑定卷軸與 Identify All

Apothecary 最常被所有玩家使用的功能，是鑑定。來源摘要提到玩家可以在這裡購買 `Identify Scrolls`，也可以回城後使用 `Identify All` 類服務一次揭示背包裡未鑑定裝備的 mods。

兩種使用方式的差異：

| 方式 | 適合情境 | 使用重點 |
| ------ | ------ | ------ |
| `Identify Scrolls` | 地城中背包快滿、需要立即決定裝備去留 | 優先用在高稀有度、高價部位或可能升級當前 build 的掉落 |
| `Identify All` | 回城後背包裡有大量未鑑定物品 | 適合地城結束後批量揭示 mods，再分成自用、分身、出售或交易 |

<note>
<p>來源摘要提到當背包中有未鑑定物品時，Apothecary 視窗可能出現 <code>Identify All</code>、全部鑑定、全て鑑定或近似按鈕。按鈕位置與名稱請以目前 UI 為準。</p>
</note>

完整鑑定流程與高價部位判斷，可接著看 [Identification 鑑定指南](nevergrind-online-identification.md)。

## Caster 裝備與商店刷新

Apothecary / Alchemist 也是 caster 向裝備商店。來源摘要提到它主要販售 `Cloth Armor` 與法系鈍器，例如單手 focus 或兩手 stave。這類裝備對 caster / healer 的價值，常不只在基礎傷害或 armor，而是在 talents、施法節奏、資源與法術傷害詞綴。

值得定期檢查的方向：

- `Cloth Armor`：適合 cloth classes，或作為低防高詞綴的 caster 補洞裝。
- Focus / stave：若出現 talents、spell damage、faster cast rate、mana / spirit sustain 或 resistance reduction，可能是很好用的過渡裝。
- 隨機刷新：來源摘要提到商店商品會依角色等級調整，約每 1 小時的角色在線時間 restock；實際週期仍以目前版本為準。
- 角色等級：商店裝備跟著角色進度變化，中期之後更值得順手看一眼。

對 `Cleric` 這類能穿 plate 但也會走 caster / healer 路線的職業來說，Apothecary 和 [Blacksmith 鍛造鋪](nevergrind-online-blacksmith.md) 都值得看：Blacksmith 提供生存與重甲底材，Apothecary 則可能提供法系武器與資源解法。

## 鑑定後的金幣效率

來源摘要提醒，裝備鑑定後若出現 talents 或高額抗性，販售價格會明顯提高。這讓 Apothecary 不只是消耗金幣的地方，也會影響你如何把地城戰利品轉成下一輪 farm 的資金。

尤其值得注意的候選：

- Focus / stave / magic blunt：若鑑定出 talents、caster 詞綴或高價 mods，可能自用或賣出好價。
- 頭盔、盾牌、胴體：若帶高抗性、talents 或生存詞綴，常有較高價值。
- Rare / Unique / Set：不只看顏色，還要看 roll 是否真的支援 build。
- 賣掉前先比較：自用、分身、交易、vendor value 四種價值可能不同。

<tip>
<p>如果背包裡有 focus / stave 這類法系底材，不要只因為顏色普通就直接賣。先鑑定看 talents 與資源詞綴，常常會比想像中值錢。</p>
</tip>

## 地城前後建議流程

1. 出發前到 Apothecary 補藥水與 `Identify Scrolls`。
2. Healer / caster 額外確認 spirit、mana 與長時間戰鬥的續航。
3. 從 [Tavern 酒館](nevergrind-online-tavern.md) 選任務並確認隊伍 ready。
4. 地城中背包快滿時，用 scroll 鑑定最有機會升級或最高價的掉落。
5. 回城後到 Apothecary 使用 `Identify All` 類服務。
6. 自用與高價候選留下，低價裝備賣掉，長期裝備再考慮 [符文 Runes](nevergrind-online-runes.md) 或升級。

總結來說，Apothecary 是 NGO 的後勤心臟之一：它讓隊伍能帶著足夠補給進地城，也讓玩家能快速把未知掉落轉成實際裝備判斷。對 healer / caster 來說，它更是維持資源、尋找法系武器與緩解續航壓力的固定巡迴點。
