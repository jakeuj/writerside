# LoL 隨機單中：大混戰（ARAM Mayhem）海克斯增幅裝置實戰整理

> 本頁整理自巴哈《【討論】隨機單中：大混戰 討論串》（第 1～15 頁）與 Blitz ARAM Mayhem Augments / Tier List，偏向實戰心得與踩雷筆記，而非完整數據百科。

## 模式簡介：隨機單中大混戰是什麼？

隨機單中：大混戰（ARAM Mayhem）是在傳統 ARAM 上疊加「海克斯增幅裝置（Augments）」與一堆混亂規則的變體：

- 無符文，改為 **等級 1 / 6 / 11** 各選一次海克斯增幅裝置。
- 近戰與遠程有額外屬性調整，部分變形英雄（豹女、傑西、蜘蛛…）會因此出現「切型態回血」等奇妙互動。
- 被控制太久會自動獲得短時間韌性／免控，避免無限連控。
- 爆水晶後每波兩隻超兵，**清線與線權**的重要性比傳統 ARAM 更高。

整體來說，這個模式是「理解英雄 + 裝備 + 海克斯三者互動」的遊戲，比誰抽到幾個稜鏡更重要。

---

## 海克斯增幅選擇大原則

1. **先看英雄機制，再看稀有度**
   - 銀／金海克斯如果剛好貼合英雄機制，常常比泛用稜鏡還強。
   - 例：普攻特效角拿到攻速／普攻觸發類銀海克斯，實戰價值可能比一顆看起來華麗但不搭配的稜鏡還高。

2. **放大你已經強的那一塊**
   - 爆擊射手 → 優先爆擊、攻速、普攻觸發增幅。
   - 技能流法師／技能係數高的 AD → 技能傷害、技能急速、基本功類。
   - 坦克／鬥士 → 生命值、回血、護盾、回血轉傷害（生死循環、蛋白飲等）。

3. **前中期目標：線權與清線**
   - 強清線的海克斯（範圍傷害、持續灼燒、龍魂類）在這模式特別實用。
   - 前期沒線權，後面爆水晶後常常連家門口都踏不出去。

4. **坦克不是不能玩，而是要「能轉傷害」的坦克**
   - 單純疊雙防會被狂堆穿甲／真傷的陣容融化。
   - 要搭配：高回血（賈克修、吸血裝）、回血增幅（蛋白飲）、回血轉傷害（死亡循環）等，讓你「越坦越痛」。

## 強勢海克斯與代表玩法

### 基本功（強化 Q/W/E 技能）

**評價：上限極高，特定英雄接近必拿。**

- 船長：
  - 桶子會吃到基本功的技能加成，甚至有玩家實測「增傷存在桶內」→ 3 連桶 > 2 連 > 單桶，傷害爆炸。
- 雷茲：
  - 搭配基本功後技能循環幾乎停不下來，加上海克斯龍魂類增幅，實戰被形容成「基本無敵」。
- 其他吃 Q/W/E 爆發的英雄：
  - 例如部分鬥士、法刺（像卡薩丁等），可以藉由改出裝走技能流來發揮基本功價值。

注意：

- 某些「靈魂在大招」的英雄（卡薩丁、石頭人…）拿基本功等於放棄大絕增幅，要搭配出裝改打法才划算。

---

### 死亡循環（治療量轉傷害）

**評價：搭配高回血角色時屬於誇張級別，札克是一哥。**

- 札克：
  - 出坦裝 + 高回血裝（賈克修、血量裝、吸血類）+ 死亡循環 → 有玩家分享 12 分鐘打出 2.5 萬額外傷害，一坨軟泥下去對面直接蒸發。
- 其他適合：
  - 高吸血鬥士（薩蜜拉、菲歐拉等）或本身技能就帶大量治療的英雄。

概念：

- 只要你「越打越回」，死亡循環就會幫你「越打越痛」，回血直接變成第二套輸出係數。

---

### 拔劍吧（遠程改近戰）

**評價：會玩的人手上是超高上限稜鏡，也是最有趣的一類。**

- 葛雷夫：拔劍 + 傲慢 + 狂躁 → 近身一發軟皮蒸發。
- 汎：拔劍 + 滾加移速海克斯 → 比易大師還難抓的跑砍怪物，抓到人三四刀蒸發，會戰後血還半條以上。
- 凱特琳：拔劍後可以近戰刺客式砍人，再切回遠程收頭，輸出空間大幅提高。
- 燼：剩一發子彈直接走到臉上「蹦」一下，爽度與威嚇感十足。

注意：

- 某些英雄技能判定會因為近遠程切換變得難用，新手不建議亂拿。

---

### 幻影武器（複製普攻特效）

**評價：需要理解普攻特效與命中效果，玩對英雄時很誇張。**

- 飛斯 W + 幻影武器：
  - 巴哈實測：存在「無限觸發到敵人死」的 Bug，特別是再疊上灼燒類裝備時，AOE 會燒到剩一人為止才停。
- 牛頭：
  - 拿幻影可以轉巫妖爆殺流，重溫舊版「W 起飛後還能普攻補一刀」打法。
- 薩科：
  - 配九頭蛇、星蝕、海妖、殞落之際等普攻特效裝，幻影武器讓他多一把看不見的刀。

延伸：

- 搭配「命中效果減 CD」類增幅（秘術衝擊）時，特效觸發頻率可以高到離譜。

---

### 終極革新（多一次大招）

**評價：理論很強，實戰有 Bug 與節奏學問。**

- Bug 與限制：
  - 約瑞科：不連續開大有機會無法正常觸發第二次。
  - 死亡重置描述寫 75 秒，但實戰常讓人覺得沒那麼快刷新，文字有誤導感。
- 實用技巧：
  - 前期一波只開一個大，死了再來一發，如此幾乎每波都有 R 可用。
  - 貪心一次丟兩發 R，常常導致下一波大招真空期偏長。

---

### 球鞋收藏家（鞋子任務）

**評價：曾經是「爆錢 Bug」，修正後多半是娛樂向稜鏡。**

- 舊版本：
  - 有玩家實測任務鞋賣掉還能拿錢，任務繼續算 → 一局等於多賺兩件大裝的經濟，堪稱生活 HAKKA。
- 目前版本：
  - 後續回報顯示賣鞋已經拿不到那筆錢，Bug 疑似修正。
  - 鞋子本身任務繁瑣、完成後強度也不算特別突出。

結論：

- 除非你喜歡解任務、有餘裕玩娛樂局，否則不算穩定強力選項。

## Bug、坑點與翻譯雷區整理

### 裝備／海克斯說明與實際不符

- 升級彎刀：
  - 說明寫鬼使彎刀、命脈系列被動冷卻減半，但玩家實測命脈系列完全沒減 CD，幾乎被公認為描述或實作有問題。
- Ornn 升級 vs 海克斯升級衝突：
  - 已用海克斯升級的裝備，如果再被厄爾升級，可能會**變回普通版效果**，等於白白浪費一個海克斯被動。

### 任務 / 疊層類問題與異常

- 心之剛（心鋼）任務：
  - 有人遇到：任務完成數字有動，但血條沒長，推測為 Bug。
  - 解法是：「任務完成後賣掉重買」，才能把加成都真正疊回身上。

### 雙型態英雄「切型態回血」

- 豹女、傑西、蜘蛛…
  - 因為近戰在這模式有額外生命加成，反覆切型態會看到血條「跳回滿」，實戰上有點像誤打誤撞的回春。

### 普攻觸發 vs 技能觸發

- 很多寫「普攻觸發」的海克斯，其實 **一定要有「普攻動畫」才算**：
  - Katarina 的匕首、部分射手法師的命中效果技能，都**不會**被當成普攻。
  - 「神射法師」在卡特身上就是典型地雷：Q/E/大全部不吃，等於浪費一顆增幅。

### 翻譯文言／敘述不清

- 例如「爆擊攻速」這類描述，中文看起來像繞口令：
  - 「你的爆擊會提供爆擊攻速」？
  - 還是「你的爆擊會增加你的攻速」？
- 實戰上多半只能靠實測或影片，建議搭配 Blitz Augments 英文版對照理解。

## 英雄代表套路與海克斯搭配

以下為討論串中出現頻率較高、實戰體感特別強的一些組合範例：

### 法師

- 雷茲：
  - 【基本功】 + 任意法強／穿透裝 + 龍魂海克斯 → 中後期幾乎是把人當紙在撕。
- 船長：
  - 【基本功】 + 爆擊（無盡、見縫插針） → 桶子傷害可以把前排一波秒到你以為是 Bug。
- 沙皇：
  - 【颱風】類海克斯 + W 沙兵 → 有玩家描述為「無限距離兩發飛彈」，清線與消耗都非常煩人。

### 刺客／鬥士

- Katarina：
  - 坦裝 + 【物攻轉魔攻】+【雙修大師】→ 大招疊滿 AP 後，一波會戰面板破 1000 AP，不論坦薄都吃不消。
  - 注意避開各種只寫「普攻觸發」的海克斯，很多實際是不吃技能的。
- Bel'Veth：
  - 過去在競技場有「穩扎穩打 Bug」的版本，現在已被修正，在大混戰模式表現相對普通。

### 射手

- 伊澤瑞爾：
  - 【彈幕神童】類疊上去後，R 幾乎會一直飛，後期每發 600–700 傷害，配上隊友控場有超強開戰與收割能力。
- 札亞、煞蜜拉、吉茵珂絲、燼、凱特、汎：
  - 共通點：拿到攻速、爆擊、拔劍、增加攻擊距離的海克斯時，都有機會變成 S 級主 C。
  - 其中拔劍汎／拔劍燼／拔劍凱特／拔劍煞蜜拉，是論壇裡討論度非常高的「暴力娛樂流」。

### 坦克

- 札克：
  - 巴哈實戰評為「撇除 Bug 後最 OP 的大亂鬥角色」，核心是【死亡循環】+ 各種生命／回血型海克斯。
- 賽恩、蒙多、瑪爾札哈、樹人等：
  - 建議出裝：先疊血量與坦度，再搭配「回血放大」或「回血轉傷害」的海克斯，定位就會從純坦變成「會打的坦」。

## 團隊節奏與觀念

1. **前中期別亂送塔，線權 > KDA**
   - 爆水晶前送一兩座外塔不算世界末日，但爆水晶後再沒兵就送塔，常常會讓遊戲直接結束。

2. **大後期的「買一送一」概念**
   - 有經驗玩家的建議：
     - 關鍵坦克或主 C 在大後期之前死掉時，至少要做到「拉著對面主 C 一起死」，確保下一波雙方火力同步重生。

3. **優勢與劣勢的打法差異**
   - 優勢方：
     - 多活久一點，穩定吃兵買裝，別因為貪塔亂衝導致連環送頭。
   - 劣勢方：
     - 輔助／坦克負責拖時間與探草，主 C 死就「一起死」，避免下一波變成 3v5／2v5。

## 如何搭配 Blitz 的 ARAM Mayhem 資訊使用？（總覽）

- Blitz Augments 頁面：
  - 提供完整海克斯清單、英文正式名稱、稀有度與數據統計（出場率、勝率）。
  - 本頁下方「Blitz Augments 結構化清單」已節錄最重要的一批稜鏡／金／銀級增幅，並加上中文暱稱與定位標籤。

- Blitz ARAM Mayhem Tier List：
  - 告訴你在目前版本中，各英雄在大混戰模式下的整體表現（S / A / B …）。
  - 建議流程：
    1. 先看自己常玩的英雄在 Tier List 的位置（高 Tier 英雄本身就有優勢）。
    2. 用本頁整理的「英雄 × 海克斯推薦表」當實戰參考，對照自己抽到的海克斯。
    3. 進遊戲選海克斯時，以「**我這英雄 + 已出裝備 + 現在抽到的海克斯，能否形成連動或 Bug 式放大**」為最高優先，而不是只看顏色。
