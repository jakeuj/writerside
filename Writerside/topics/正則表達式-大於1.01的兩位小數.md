正則表達式 大於1.01的兩位小數

```

^1\.0[1-9]$|^\.[1-9][0-9]*$|^[2-9](\.[0-9]{1,2})*$|^[1-9][0-9]+(\.[0-9]{1,2})*$
```

參照：

http://www.runoob.com/regexp/regexp-syntax.html

同場加映 Match => t=TTT,p=PPP

```

let origin = '<t>TTT,<p>PPP';
let matchArray = origin.match(/<t>(.+),<p>(.+)/);
let t = matchArray[1];
let p = matchArray[2];
```
