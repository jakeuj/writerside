# Nevergrind Online Blacksmith 鍛造鋪指南

Nevergrind Online 的 `Blacksmith` 鍛造鋪 / 鐵匠位於 Edenburg 右側，是重裝武庫，也是後期裝備客製化的工坊。想買 plate / mail armor、重型近戰武器、處理裝備升級，或把 runes 放進 socketed items，都會回到這裡；真正要投入高價符文前，先確認裝備會穿很久、roll 夠好、sockets 值得，會比事後心痛便宜很多。

- 檢視日期：`2026-05-03`
- 分類：[Edenburg 城鎮設施總覽](nevergrind-online-edenburg.md)
- 資料來源：NotebookLM 摘要、Nevergrind Wiki / SteamDB patch notes、[Nevergrind Online 攻略DB：クラフト](https://atelier3.web.fc2.com/ngo/mythical.html) 與來源資料整理
- 版本提醒：Blacksmith 的販售品類、裝備升級、enchanting / crafting counter、rune upgrade 與符文是否可取回，可能因版本更新而調整；實際操作請以目前遊戲內 UI 與 tooltip 為準

<tldr>
<p>Blacksmith 是 plate / mail armor、重型武器與後期裝備強化的核心設施。</p>
<p>Runes 鑲嵌與 Craft tab 的 Mythical / Rune Words 都在 Blacksmith 相關功能處理，但兩者規則不同。</p>
<p>Plate 類重裝通常有很高的原始價值，鑑定後若出現 talents 或高抗性，也可能成為重要金幣來源。</p>
</tldr>

## Blacksmith 在 Edenburg 的定位

Edenburg 裡不同設施負責不同裝備層級。[Apothecary](nevergrind-online-apothecary.md) 偏補給、鑑定與法系需求，`Merchant` 偏背包與一般商品，`Blacksmith` 則面向重型防具、近戰武器與後期強化。

| 設施 | 裝備 / 功能重點 | 使用情境 |
| ------ | ------ | ------ |
| `Blacksmith` | `Plate`、`Mail`、重型武器、item upgrades、runes / crafting | 重裝職業補裝、長期裝備強化、socketed items 客製化 |
| [Apothecary](nevergrind-online-apothecary.md) | 藥水、鑑定卷軸、`Identify All`、偏法系或布甲需求 | 地城後補給、批量鑑定與 caster 方向整理 |
| `Merchant` | 背包擴展、一般商品、部分輕型防具 | 早期生活品質、背包管理與一般購買 |

對 `Warrior`、`Crusader` / `Paladin`、`Shadow Knight`、`Cleric` 這類能穿 plate 的角色來說，鍛造鋪不只是商店，而是每次推高難度前確認物理防禦、抗性與重武器替換的地方。

## 重甲與重型武器

來源摘要把 Blacksmith 視為高等級物理裝備的重要供應點。它偏向販售外觀與定位都較重的裝備，例如 `Plate Armor`、`Mail Armor` 與重型近戰武器。

常見判斷：

- `Plate`：物理防禦與基礎價值高，適合 plate classes 與容易被打到的位置。
- `Mail`：介於重甲與輕甲之間，適合 `Ranger`、`Rogue`、`Shaman` 等 mail classes 或過渡需求。
- 重型武器：適合近戰物理角色補 weapon base、傷害或 proficiency 方向。
- 高等級底材：如果 quality tier、等級與詞綴方向都對，可能比顏色本身更值得關注。

Blacksmith 販售品通常和角色等級、隨機詞綴與 restock 節奏綁在一起。來源摘要整理為約每 1 小時的角色在線時間刷新一次；實際週期仍以目前 UI 為準。這代表它不只是「買固定裝備」的地方，而是刷寶之外的小型補裝池：

- 等級貼近當前角色時，先看 plate / mail / heavy weapon base 是否能補防禦或傷害缺口。
- 出現 talents、高抗性、主屬性、防禦或 sockets 時，值得和地城掉落一起比較。
- 買裝前先確認 gold 是否該留給 skills、inventory upgrade、補給或 [Gambling 賭博](nevergrind-online-gambling.md)。

<note>
<p>職業可穿防具與武器 proficiency 仍要以角色本身為準。完整職業裝備可用性可看 [Nevergrind Online 職業系統總覽](nevergrind-online-classes.md)。</p>
</note>

## Runes 與後期客製化

Blacksmith 最容易被後期玩家記住的功能，是 `Runes`、enchanting 與 crafting 相關 counter。SteamDB patch notes 曾提到 Blacksmith 的 enchanting counter 啟用，以及後續 crafting counter 加入 rune upgrade；因此，鍛造鋪已經不只是販售裝備，而是讓 socketed items 進入後期客製化的核心節點。

基本操作邏輯很單純：先拿到有 `Sockets` 的裝備，再到 Blacksmith 相關功能把 rune 放進去。難的是判斷「這件裝備是否值得吃掉這顆 rune」，因為來源摘要把鑲嵌視為永久投入。

常見強化方向可以先這樣理解：

| Rune 方向 | 常見用途 |
| ------ | ------ |
| Weapon damage / attack ability | 提升物理武器輸出、命中或穩定度 |
| Enemy resistance reduction | 讓 caster / hybrid 更容易打穿高抗性目標 |
| Crit / haste / attack speed | 放大爆發窗口或加快 rotation |
| Resist / armor / life | 補 Hell 難度生存缺口 |
| Magic find / rare drop | 用在安全 farm 或刷寶裝，犧牲部分戰鬥性能換收益 |

鑲嵌或升級前先確認：

1. 這件裝備是否真的有 sockets。
2. 這件裝備是否值得長期穿，不只是過渡裝。
3. Rune 是否適用於該部位，例如 weapon、armor、charm 或特定 1h / 2h 類型。
4. 目前版本是否允許取回、覆蓋、升級或替換 rune。
5. 這顆 rune 是補 build 缺口，還是只是讓數字看起來漂亮。

<warning>
<p>來源摘要把 rune 鑲嵌視為永久且不可逆。即使目前版本已有 crafting / upgrade 類功能，也建議先把高階 rune 當成慎重資源：先查 tooltip，再點確認，不要把頂級 rune 塞進很快會換掉的過渡裝。</p>
</warning>

完整 rune 取捨、物理 / 魔法 / 防具方向，可看 [符文 Runes 指南](nevergrind-online-runes.md)。

## Item upgrades 與裝備投入順序

Nevergrind Wiki 的 item upgrade 資料把城鎮升級描述成穩定強化裝備的方式。實際 UI 可能會和 Blacksmith、crafting 或其他 counter 串在一起；無論入口名稱如何，升級前的原則都相同：不要把大量金幣或資源砸在很快會換掉的裝備上。

比較穩的投入順序：

1. 先升級核心技能、背包與補給能力。
2. 確認目前最拖累角色的是 weapon、armor、resists 還是資源。
3. 只對會穿一段時間的裝備做 upgrades。
4. 若裝備同時有好 roll、好 sockets 與正確部位，再考慮 runes。
5. 高階 rune 與 crafting 投入前，再和 unique / set / legendary 候選比較。

## Crafting、Mythical 與 Rune Words {#crafting-mythical-rune-words}

[Nevergrind Online 攻略DB 的 Craft 頁](https://atelier3.web.fc2.com/ngo/mythical.html) 把 Blacksmith 的 `Craft` tab 描述成製作 `Mythical items` 的地方，也可理解成 `Rune Words`。這套系統和前面提到的「把 rune 直接鑲進 socket」不同：Craft 需要的是 base item 與配方要求的 runes 都在背包裡，選好 base 後按 craft，不需要先把 runes 放進裝備 socket。

SteamDB 的 patch notes 也能補上這套系統的脈絡：Blacksmith 先啟用 rune enchanting，後續才逐步開放 socketed base 的特殊屬性、`Superior` base、Craft counter、Mythical recipes 與 rune upgrade。這代表 crafting 不是單純「照表放符文」，而是把版本、base item、recipe 與 UI 狀態一起檢查的後期系統。

### Patch notes 補充的版本演進

| 日期 | 更新重點 | 筆記上的判斷 |
| ------ | ------ | ------ |
| 2024-10-24 | Blacksmith 的 enchanting counter 啟用，socketed items 可以吃 rune bonus，charms 也開始能有 sockets | 一般 rune socketing 和 Mythical crafting 要分開理解 |
| 2025-04-13 | Socketed items 開始能掉出 special properties，且成功 craft Mythical item 後會保留這些屬性 | 好 base 的價值不只在 sockets，也在自帶 mods |
| 2025-04-14 | `Superior` items 加入，craft 後可依 base 的特定屬性鎖定職業方向，並避免重複指派同一個 talent bonus | `Superior`、職業 talents 與 skill enhancements 都是挑 base 的重點 |
| 2025-04-16 | Crafting counter 先以單一測試 item 啟用，並調整 socketed item special props 與 shield socket 上限 | Crafting 是分階段開放過的系統，socket 上限與可用配方可能會變 |
| 2025-04-22 | Normal / socketed items 能出現在 vendors，後續又修正 previously enchanted items 也可 craft 的 bug | Vendor 巡店是找 base 的來源；已 enchant 裝備不應視為正常 craft base |
| 2025-05-17 | Season 3 多種 Mythical weapons 完成，crafted item 會正確採用 base item 的等級需求 | 成品等級要看 recipe 和 base 兩邊，不要只看 blueprint |
| 2025-05-20 | Crafting counter 加入 rune upgrade，並把 UI 改成用 item icons 選 blueprint 類別 | Craft tab 同時關聯 Mythical recipes 與 rune upgrade，實際操作要看目前 UI |
| 2025-05-29 | Season 3 的 crafting counter 開放所有 recipes | 舊版「測試 / preview」資訊只能當歷史脈絡，實際 recipes 以當前版本為準 |
| 2025-10-07 | 修正 craft rune 後成本顯示錯誤 | 涉及 rune upgrade / crafting cost 時，一律以當前 UI 成本為準 |

### Craft 基本流程 {#craft-basic-flow}

1. 到 Edenburg 右側的 Blacksmith，切到 `Craft` tab。
2. 背包中準備一件符合 recipe 的 socketed base item。
3. 背包中準備 recipe 要求的 runes。
4. 選取 base item 後執行 craft。
5. 成品會變成 Mythical / Rune Words 類裝備；後續能否再改造，請以目前 UI 顯示為準。

<note>
<p>Craft 使用的 runes 不需要預先鑲嵌。這點和一般 rune socketing 很容易混淆：Craft 是配方消耗素材，一般 socketing 則是把 rune bonus 直接放進裝備。</p>
</note>

### Base item 要求

Craft 的關鍵不是只湊到 rune，而是找到正確 base。

| 規則 | 判斷方式 |
| ------ | ------ |
| 名稱顏色 / 狀態 | 使用 socketed item，也就是名稱灰色或帶 sockets 的白色裝備 |
| 孔數 | sockets 數必須剛好等於 recipe 要求的 rune 數；多一孔或少一孔都不合格 |
| 原有 mods | base 上既有 talents 或 skill enhancements 會被繼承 |
| `Superior` | 若 base 有 `Superior` 類前綴，武器傷害或防具物理防禦加成會保留並和成品屬性疊加 |
| `Ethereal` | ethereal 狀態會繼承，但 craft 不會恢復耐久度 |
| 材質 / 底材 | 成品的材質與基礎性能依 base 而定，例如 cloth base 做出來仍是 cloth armor |

這些規則讓 base item 本身變得非常重要。高 sockets 只是門票；真正值得 craft 的 base，還要看 talents、skill enhancements、Superior、Ethereal、基礎傷害 / 速度 / 防禦與職業可用性。

### Craft 後的規則 {#craft-after-rules}

來源頁整理的 craft 後規則可以先記成這幾點：

- Craft 會消耗 item sockets，成品不能再把剩餘孔位拿去一般 enchant / socketing。
- Craft 使用的 runes 不會把各自的單獨鑲嵌效果加到成品上；成品吃的是 recipe 給的屬性。
- 成品要求等級取 base level 與 recipe level 的較高者。
- base 的基本性能會繼承，例如 weapon damage、speed、armor / physical defense。
- 若 base 帶有特定職業 talents 或 skill enhancements，成品的隨機 mod 會鎖定在該職業方向。
- base 上已存在的 talents 不會和 craft 追加 talents 重複，會從抽選中排除；skill enhancements 則可能和 craft 追加項目重疊。

<warning>
<p>Craft 後 sockets 會被視為已消耗，而且 rune 的單獨效果不會附加到成品上。不要把 Mythical / Rune Words 當成「先拿 rune bonus，再額外拿 recipe bonus」的雙重收益。</p>
</warning>

### Base 來源與商店巡查

Socketed base item 可以在地城裡掉落，也可能出現在 Apothecary、Blacksmith 或 Merchant 的商品裡。來源頁對 restock 時機使用了帶保留的說法，整理為城鎮或離線狀態經過約 1 小時後可能刷新；實際仍以目前版本 UI 為準。

巡店時可以這樣看：

- Apothecary：留意 cloth、focus、stave 類 caster / healer base。
- Blacksmith：留意 plate、mail、heavy weapon base。
- Merchant：留意 leather、一般部位與可能剛好符合 recipe 的 socketed base。
- 任何 base 都要確認 sockets 數是否剛好符合目標 recipe。

換句話說，Blacksmith 的 Craft tab 不是只看 rune 清單，而是把「正確 base + 正確 rune recipe」合成成 Mythical 裝備。好 base 的價值會因此大幅上升。

### Craft 卡住時檢查 {#craft-troubleshooting}

如果 Craft tab 的按鈕沒有亮、base 不符合 recipe，或做出來的結果和預期不同，先用這張表排查：

| 症狀 | 先檢查 |
| ------ | ------ |
| `Craft` 按鈕沒有反應 | Gold、recipe 要求的 runes、選到的 base type、sockets 數是否都符合目前 blueprint |
| 找不到可用 base | 先看地城掉落、Apothecary、Blacksmith、Merchant 的 normal / socketed items；socket 數必須剛好等於 recipe rune 數 |
| 想把已鑲嵌 rune 的裝備拿去 craft | SteamDB patch notes 曾把 previously enchanted items 可被 craft 視為 bug；不要把已 enchant 裝備當作正常 base |
| 成品 required level 比想像高 | Crafted item 會取 recipe level 與 base item level 的較高者 |
| Mythical tooltip 沒顯示 total sockets | 2025-05-20 patch notes 調整過 Mythical / crafting descriptions 的 socket 顯示方式 |
| Rune upgrade 成本看起來怪 | 2025-10-07 patch notes 曾修正 craft rune 後的 cost 錯誤；以目前 UI 顯示為準 |

更完整的來源分級、[快速製作指南](nevergrind-online-blacksmith-crafting-recipe-research.md#crafting-quick-guide)、[素材選擇優先級](nevergrind-online-blacksmith-crafting-recipe-research.md#crafting-base-selection)、[屬性繼承規則](nevergrind-online-blacksmith-crafting-recipe-research.md#crafting-inheritance)、官方已確認 Mythical families、素材 farming 與版本風險，可看 [Blacksmith Crafting / Recipe 深度筆記](nevergrind-online-blacksmith-crafting-recipe-research.md)。

## 金幣與回收價值

來源摘要提到，plate 類重裝的原始價格通常很高，外觀看起來越重的裝備往往越值得留意。這讓 Blacksmith 相關裝備不只對穿重甲的角色有用，也對所有玩家的金幣流轉有價值。

地城後的整理策略：

- 未鑑定的 plate、mail、重型武器，如果等級與底材不差，通常值得優先帶回。
- 先到 [Identification 鑑定指南](nevergrind-online-identification.md) 介紹的流程揭示 mods。
- 若鑑定出 talents、高額抗性、主屬性、防禦或高價值詞綴，先比較自用、分身、交易與出售。
- 若詞綴不合 build，但 vendor value 高，可以當作穩定金幣來源。

<tip>
<p>背包滿時，重裝不是一定要全撿，但 plate / mail / high-level weapon base 值得比低價低等部位多看一眼。它們常常是「不自用也能換錢」的候選。</p>
</tip>

## 和其他設施搭配

Blacksmith 的價值通常不是單獨發生，而是和其他城鎮流程串起來：

1. 從 [Tavern 酒館](nevergrind-online-tavern.md) 選任務，依掉落偏好 farm 裝備。
2. 地城後回 [Apothecary](nevergrind-online-apothecary.md) 鑑定與補給。
3. 把 plate、mail、重武器、socketed items 交給 Blacksmith 方向評估。
4. 有長期價值的裝備才投入 upgrades、runes 或 crafting。
5. 沒有 build 價值但售價高的重裝，轉成金幣支援下一輪 farm。

對穿 plate 的 `Cleric` 來說，Blacksmith 尤其關鍵：它同時影響生存、物理防禦、抗性補洞與後期 rune plan。對其他職業來說，它也代表裝備客製化與高價戰利品回收的核心工坊。
