# Nevergrind Online Magic Items 魔法裝備指南

Nevergrind Online 的藍色 `Magic` 裝備是玩家最常接觸到的強化掉落物，通常只有 1 到 2 個隨機 mods。它們不是單純的低階垃圾：來源摘要特別指出，magic 單一詞綴有時能 roll 得比同等級 rare 更高，因此在補抗性、資源或特定缺口時，藍裝仍然可能是很精準的解法。

## Magic 在稀有度體系中的位置 {#magic-rarity-position}

藍色 `Magic` 是隨機裝備的基礎型態。它不像黃色 `Rare` 追求多條詞綴同時命中，也不像紫色 `Unique` 或綠色 `Set` 靠固定效果成形；magic 的判斷重點是「少數詞綴是否剛好補到當下最痛的缺口」。

| 類型 | 詞綴結構 | 判斷方式 |
| ------ | ------ | ------ |
| `Magic` | 1 到 2 個隨機 mods | 看單一詞綴是否夠高、是否剛好補短板 |
| `Rare` | 2 到 6 個隨機 mods | 看多個詞綴是否同時支援 build |
| `Unique` | 固定 mods，數值浮動 | 看固定效果、技能加成、部位與 roll |
| `Set` | 固定 mods，加上多件 bonus | 看 bonus 是否值得犧牲其他部位 |

所以 magic 的定位不是「一定很弱」，而是「窄，但可能很尖」。當角色缺某一種抗性、生命、資源、施法節奏或特定功能時，一件詞綴正確的藍裝可以比詞綴分散的黃裝更直接。

## 單一詞綴的價值

來源摘要中最重要的觀察是：magic 物品的單一屬性數值，有時會比同等級 rare 更高。這讓藍裝在裝備篩選上有一個很明確的角色：不要拿它和六詞綴裝備比總量，而是看它是否能把某個門檻一次補滿。

常見用途：

- 抗性填補：火、冰、毒、血、魔法或狀態抗性不足時，高 roll 藍裝可能立刻改善生存。
- 資源補洞：mana、spirit、leech 或回復不足時，用藍裝暫時穩住 rotation。
- 施法或攻擊節奏：若剛好出現 faster cast rate、attack haste 或相關節奏詞綴，可以當作過渡工具。
- 特定賣價：某些高價部位若 roll 到 talents 或高抗性，即使不自用也可能有不錯的 vendor value。

<tip>
<p>判斷 magic 時，先問它有沒有「一眼就能解決的問題」。如果它只是兩條普通詞綴，通常不值得久留；如果它讓你少死、少缺資源或剛好過抗性門檻，就值得多看一眼。</p>
</tip>

## 什麼時候該留藍裝

藍裝最適合在裝備還沒成形、或高階裝備有明顯缺口時使用。尤其是 early / mid game，角色常常不是缺整套完美裝，而是缺一個讓地城能穩定打完的門檻。

優先保留或鑑定的情況：

- 目前最缺的抗性剛好被高數值補上。
- 角色因 mana / spirit / life sustain 不穩，藍裝剛好補到資源問題。
- 部位本身很缺替代品，例如 head、chest、shield、focus / stave 或特定飾品。
- 詞綴雖少，但其中一條直接支援主要技能、talents、cast speed、attack speed 或防禦門檻。
- 你正在拆 set 或替換 unique，需要一件短期補洞裝撐過新的配置。

可以較快放棄的情況：

- 詞綴沒有補到任何當前缺口。
- 數值普通，且同部位已有 rare、unique、set 或 legendary 類候選。
- 等級與 quality tier 太低，基礎 armor / damage 已經跟不上。
- 裝備部位或武器類型不符合職業 proficiency。

## 掉落與鑑定經濟

來源摘要把 magic 放在很常見、但仍值得篩選的掉落層。擊敗 `Champion Monsters` 時，通常至少會掉 1 件藍色或更好的物品；`Unique Monsters` 與 boss 則通常至少掉 2 件藍色或更好的戰利品。這讓藍裝成為刷地城時最常進入背包、也最需要快速判斷的類別；完整操作可搭配 [Identification 鑑定指南](nevergrind-online-identification.md)。

比較值得帶回鑑定的藍裝部位：

- Magic blunt、focus、stave：若出現 talents、spell damage、施法速度或相關 caster 詞綴，可能有自用或售價價值。
- Chest：基礎防禦、生命、抗性與高價值詞綴都值得看。
- Shield：block、all res、status res、magic damage reduction 或防禦詞綴可能很實用。
- Head：talents、主屬性、magic find、抗性或資源詞綴可能讓它比普通黃裝更好用。

<note>
<p>藍裝掉得多，不代表每件都該鑑定。背包空間和鑑定成本也是資源；把注意力集中在高價部位、當前缺口與可能出 talents / 高抗性的裝備上，效率會好很多。</p>
</note>

## 和 Rare / Unique / Set 的取捨

Magic 和 [Rare Items](nevergrind-online-rare-items.md) 的差異，主要是「單點」對「組合」。Rare 有機會靠 2 到 6 個 mods 組成完整解法；magic 則是在更少詞綴裡追求更高的單點補強。

和 [Unique Items](nevergrind-online-unique-items.md) 或 [Set Items](nevergrind-online-set-items.md) 比較時，magic 通常不會負責 build 核心，但它很適合補固定裝沒照顧到的洞。例如你換上強力 unique weapon 後抗性掉太多，或為了維持 set bonus 犧牲了資源恢復，一件好藍裝就能暫時把角色拉回安全線。

常見判斷：

- 如果你缺的是單一抗性門檻，magic 可能比普通 rare 更有效。
- 如果你缺的是整體輸出、生存與資源的多項平衡，rare 通常更值得期待。
- 如果 unique / set 的固定效果能改變 rotation，magic 多半只能補洞，不能取代核心。
- 如果 magic 的單一詞綴高到剛好解決高難度地城問題，它就有保留價值。

## 判斷一件 Magic 值不值得留 {#magic-keep-checklist}

1. 它是否補到目前最缺的抗性、資源、生存或節奏問題。
2. 單一詞綴是否明顯高於你現有裝備或同階 rare 候選。
3. 部位是否本來就難找替代品，或是 vendor value 較高。
4. Quality tier 與等級是否足夠撐到下一階段。
5. 它是否讓你可以順利替換 set、unique 或 weapon 配置。
6. 若不是自用，是否值得放進銀行、給分身或賣掉。
