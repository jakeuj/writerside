# Nevergrind Online FC2 職業 Build 摘要

FC2 的 14 個職業頁比較像高階玩家的 build 讀法：它會列出職業定位、套裝路線、talent 例、裝備例與立回，而不是新手從 1 級照抄的完整教學。這篇把本地下載的職業頁逐一抽出成公開筆記可用的摘要；真正配點、交易或鑲 rune 前，仍要回遊戲內 tooltip、目前版本與自己的隊伍配置確認。

- 檢視日期：`2026-05-05`
- 分類：[職業系統與裝備可用性總覽](nevergrind-online-classes.md)
- 延伸閱讀：[FC2 各職代表技能速查](nevergrind-online-fc2-signature-skills.md)、[FC2 各職嚴選 Unique 裝備速查](nevergrind-online-fc2-selected-unique-items.md)、[套裝（Set Items）指南](nevergrind-online-set-items.md)
- 資料來源：[FC2 攻略 DB：職業頁](https://atelier3.web.fc2.com/ngo/)（本次整理本地鏡像中的 14 個 class HTML）
- 版本提醒：FC2 是玩家 meta snapshot；職業名稱、skill rank、裝備例與高階判斷可能落後目前遊戲版本

<tldr>
<p>FC2 職業頁的重點不是「照表畢業」，而是看每個職業在高階內容裡通常靠什麼技能、套裝與裝備 slot 成形。</p>
<p>看職業頁時先讀職業節奏，再看套裝路線，最後才查裝備例中的 Unique / Set / Legendary 名字。</p>
<p>若裝備例反覆出現 <code>Cryptic Paragon</code>、<code>Charlatan's Crest</code>、<code>Zamtil's Plenitude</code>、<code>Marshal Gauntlets Iniquity</code> 或高 socket bow，優先把它當成跨職業關鍵字，而不是單一職業專屬結論。</p>
</tldr>

## FC2 明列的 rank / breakpoint {#fc2-class-rank-breakpoints}

這些數字只代表 FC2 頁面在本次鏡像中的玩家整理，適合拿來提醒「裝備加 talent / skill rank 時要回頭核對 breakpoint」。不要把它們寫成官方永久公式。

| 職業 | FC2 頁面明列的 rank 提醒 | 讀法 |
| ------ | ------ | ------ |
| Warrior | `Rupture` rank 10 / 15 / 23 / 31；`Whirlwind` rank 14 / 20 / 28；`Double Throw` rank 19 / 29；`Furious Cleave` rank 18 / 24 / 32 | 裝備 talent boost 要配合 hit count / 次數增加點調整，不是越平均越好。 |
| Crusader | `Consecrate` rank 14 / 20 / 28；`Zealous Slam` rank 19 / 29 | 物理 / holy route 裝備要看是否把核心 AoE 或單體循環推到下一段。 |
| Monk | `Superior Chakra Blast` 的 `Hadoken` cooldown：rank 17 約 9.5s、rank 26 約 5.6s；`Superior Hadoken` 的 `Dragon Punch` cooldown：rank 17 約 12.6s、rank 26 約 9.2s；`Superior Dragon Punch` 的 `Tiger Strike` cooldown：rank 19 約 6.6s、rank 28 約 2.9s；`Mastery: 千の拳` 的 `Hurricane Kicks` cooldown：rank 10 約 11.6s | Monk 裝備與 talent 不是只看傷害，也要看 cooldown 壓到哪一段，否則 rotation 可能仍然卡。 |
| Enchanter | `Superior Enthrall` 在 rank 19 / 29 增加 bonus hit；`Superior Mind Blitz` 在 rank 19 / 29 增加 bonus hit；`Mastery: 誘惑` 的 arcane reduction 表從 rank 15 到 34 約對應 37% 到 50% | Enchanter 的 breakpoint 牽動隊伍總輸出；尤其是 haste、arcane reduction、Mind Blitz 目標數與物理隊伍加成。 |

## 讀職業頁時的流程 {#fc2-class-reading-flow}

1. 先看 FC2 對職業節奏的描述，確認它是偏 solo、party support、burst、DoT、tank 還是 farm。
2. 對照該職三套 set：Normal 用來理解早期方向，Exceptional 常是中期骨架，Elite 才是長期目標。
3. 再看 `各職の代表スキル`，確認裝備上的 tree / talent / skill roll 是否真的命中核心。
4. 查看裝備例中的跨職業關鍵字，例如 Haniwa、Shako、Zamtil legs、高 socket bow 或 rare amulet。
5. 把 FC2 的「強」改寫成「這件解決了什麼問題」：breakpoint、抗性、資源、attack ability、rare drop、undead damage、caster speed 或 team utility。
6. 最後回遊戲內 tooltip，確認 rank、socket、roll range、可穿防具與目前版本是否一致。
