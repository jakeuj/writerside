# Nevergrind Online 套裝 Set Items 指南

Nevergrind Online 的套裝物品以綠色文字顯示，單件通常有預設詞綴，穿上同一套裝的多個部位後會啟用額外的 `Set Bonus`。看套裝時不要只看單件數值；真正的價值常在 2 件、3 件或特定部位組合能不能補上 build 的關鍵缺口。

- 檢視日期：`2026-05-03`
- 分類：[Nevergrind Online 物品與戰利品總覽](nevergrind-online-items-loot.md)
- 資料來源：Swiftyhorn / 高階玩家指南相關來源摘要
- 版本提醒：套裝名稱、bonus、部位與數值可能因版本不同而調整；交易、賭博或投入 rune 前，請先用目前遊戲內 tooltip 核對

<tldr>
<p>Set item 是綠色裝備；單件看固定 mods，多件才看 set bonus。</p>
<p>每個職業大致有 Normal、Exceptional、Elite 三階套裝。</p>
<p>有些低等套裝部位會因 sockets、magic find 或特定 bonus，在後期仍有混搭價值。</p>
</tldr>

## 套裝機制

套裝裝備的基本判斷可以拆成三層：

1. 單件屬性
   - 看部位、等級、職業可用性、基礎防禦 / 傷害與固定 mods。
2. 套裝件數
   - 穿同一套裝的多個部位後，會啟用額外 `Set Bonus`。
3. 混搭價值
   - 高階玩家不一定穿滿一整套；有時 2 件或 3 件 bonus 加上其他 unique / legendary，反而更適合 farm 或輸出路線。

<note>
<p>套裝的詞綴種類通常較固定，但數值仍可能有 roll range。看到同名套裝件時，仍要比較數值、socket、裝備需求與 bonus 是否接近目標。</p>
</note>

## 值得注意的套裝部位與組合

下面是來源摘要中特別被提到的套裝件或組合。這些名字適合當 farm、trade、gambling 或銀行保留的關鍵字。

| 名稱 / 組合 | 重點 | 使用情境 |
| ------ | ------ | ------ |
| `Demetrium's Ballista` | level 29 set bow，來源摘要提到有 6 sockets | 即使等級低，也可能因 rune space 成為物理職後期副手 / ranged slot 候選 |
| `Zamtil` 3 件 | 來源摘要提到 33% magic find 與 13% arcane 強化 | `Enchanter` elite 套裝；法系 farm 或混搭 magic find 候選 |
| `Nylith` 2 件 | 來源摘要提到 +25 `Stamina`，常選 cloak / boots 類部位 | `Druid` elite 套裝；追求生存與高難度容錯時可考慮 |
| `Daahoud` 3 件 | 來源摘要提到 30% magic find 與 125 attack | `Monk` elite 套裝；也可能被戰士當成過渡或 farm 混搭 |
| `Zarth` 3 件 | 來源摘要提到 +15 archery 與 +40 critical hit | `Ranger` exceptional 套裝；弓術路線的重要關鍵字 |
| `Falzain` 3 件 | 來源摘要提到 45% magic find，常用 3 件組合觸發 | `Templar` exceptional 套裝；以 magic find 混搭為主要賣點 |

## 收集與整理小提示

- 屬性波動：同名裝備仍可能有不同數值 roll；追求 perfect roll 是後期長線目標。
- Gambling：來源摘要提到城鎮右側商人可用 gold gambling；鎖定特定圖示可能是補套裝部位、尤其是飾品類的方式之一。
- 鑑定：撿到大量裝備後，可回城找 `Apothecary` 使用 `Identify All` 類功能；實際按鈕名稱以目前 UI 為準。
- 銀行共享：帳號內共享銀行很適合存其他職業套裝，留給分身角色或未來 build。
- 不要只追滿套：如果 2 件或 3 件 bonus 已經達成目標，剩餘部位可以留給 unique、legendary 或高 socket 裝。

## 和其他裝備系統怎麼接

套裝的價值常和其他系統綁在一起：

- 要先理解綠裝與 set bonus 的定位，可以回到 [物品與戰利品總覽](nevergrind-online-items-loot.md)。
- 若套裝部位有 sockets，接著看 [符文 Runes 指南](nevergrind-online-runes.md)，判斷是否值得投入 rune。
- 若要比較同部位高階裝，可以搭配 [Legendary Items 清單筆記](nevergrind-online-legendary-items.md) 查候選裝備。
- 牧師或 caster build 若提到 `Tunso`、`Zamtil`、`Falzain` 這類名字，先確認 bonus 是否真的服務於目前 rotation。
