# Nevergrind Online 套裝（Set Items）指南

Nevergrind Online 的套裝物品以綠色文字顯示，單件通常有預設詞綴，穿上同一套裝的多個部位後會啟用額外的 `Set Bonus`。看套裝時不要只看單件數值；真正的價值常在 2 件、3 件或特定部位組合能不能補上 build 的關鍵缺口。

- 檢視日期：`2026-05-05`
- 分類：[Nevergrind Online 物品與戰利品總覽](nevergrind-online-items-loot.md)
- 資料來源：Swiftyhorn / 高階玩家指南相關來源摘要、FC2 / atelier3 攻略 DB set、unique、loot、gambling 與職業頁
- 版本提醒：套裝名稱、bonus、部位與數值可能因版本不同而調整；交易、賭博或投入 rune 前，請先用目前遊戲內 tooltip 核對

<tldr>
<p>Set item 是綠色裝備；地位接近 unique，但更強調多件搭配後的 set bonus。</p>
<p>每個職業大致有 Normal、Exceptional、Elite 三階套裝。</p>
<p>高階配裝常不是穿滿全套，而是在 set bonus、unique / legendary 單件與 sockets 之間取捨。</p>
</tldr>

## 套裝機制

在 rarity 架構裡，綠色 set 和紫色 unique 很像：單件都有比較穩定、可預期的詞綴方向。但 set 的特色不在「單件一定比較強」，而是在多件同套裝備疊到指定件數後，會給角色一段穩定且可預期的屬性飛躍。

套裝裝備的基本判斷可以拆成三層：

1. 單件屬性
   - 看部位、等級、職業可用性、基礎防禦 / 傷害與固定 mods。
2. 套裝件數
   - 穿同一套裝的多個部位後，會啟用額外 `Set Bonus`。
3. 混搭價值
   - 高階玩家不一定穿滿一整套；有時 2 件或 3 件 bonus 加上其他 unique / legendary，反而更適合 farm 或輸出路線。

常見 set bonus 方向包含：

- 大幅屬性提升，例如 `Stamina`、attack、critical hit。
- 技能或 proficiency 強化，例如 archery、talent 或特定法系方向。
- 稀有功能性加成，例如 magic find。
- 補足 build 的生存、命中、施法節奏或 farm 效率缺口。

<note>
<p>套裝的詞綴種類通常較固定，但數值仍可能有 roll range。看到同名套裝件時，仍要比較數值、socket、裝備需求與 bonus 是否接近目標。</p>
</note>

## 套裝階級與選擇策略 {#set-selection-strategy}

職業套裝表適合當成「從哪一套開始看」的地圖，不是終局配裝清單。實戰上要把套裝階級、件數 bonus、部位成本和可替代的 unique / legendary 一起比較。

| 階級 | 常見定位 | 選擇重點 |
| ------ | ------ | ------ |
| `Normal` | 前期過渡、低等特殊工具裝 | 通常不用追滿套；若單件有特殊 socket roll 或功能性才值得長留 |
| `Exceptional` | 中期成形與高性價比 bonus | 很多 build 會先湊 2 件或 3 件，把核心傷害、生存或 farm 缺口補起來 |
| `Elite` | 後期長期目標 | 是終局配裝骨架，但常會拆掉弱勢部位，換成更強 unique、legendary 或高 socket 工具裝 |

最穩的讀法是先找「最小有效件數」。如果 2 件 bonus 已經補到路線需求，剩下部位就可以拿去比較 `All Talents`、抗性、casting haste、weapon base、magic find 或 rune space。穿滿套只有在最後幾件 bonus 明顯大於替代裝備時，才是值得追的結論。

## 值得注意的套裝部位與組合 {#set-notable-pieces}

下面是來源摘要中特別被提到的套裝件或組合。這些名字適合當 farm、trade、gambling 或銀行保留的關鍵字。

| 名稱 / 組合 | 重點 | 使用情境 |
| ------ | ------ | ------ |
| `Fanatic` 2 件 | FC2 `Fanatic` set 頁列出 2 件 `+15% damage vs undead`，3 件再給 `+15% damage vs demon` | 十字軍（Crusader） exceptional 套裝；中期打 undead / demon 路線時，2 件 bonus 很容易成為高性價比選項 |
| `Demetrium's Ballista` | level 29 set bow；FC2 set 頁列為 `Socketed (1-6)`，FC2 loot 頁也把最高 socket roll 列為 Normal bow 帶回城例外 | 若 roll 到高 sockets，即使等級低，也可能因 rune space 成為物理職後期 ranged slot 工具裝 |
| `Sinifay's Golden Harrier` | Ranger elite set bow；FC2 `Rune Select` 頁拿它當 Ranger bow rune 例子 | 較慢的高階 set bow；比較 `Cros`、`Rok`、`Targ` 時，要看 base speed、socket count 與目前 tooltip |
| `Zamtil` 3 件 | 來源摘要提到 33% magic find 與 13% arcane 強化 | 恩路者 / 幻術師（Enchanter） elite 套裝；法系 farm 或混搭 magic find 候選 |
| `Nylith` 2 件 | 來源摘要提到 +25 `Stamina`，常選 cloak / boots 類部位 | 德魯伊（Druid） elite 套裝；追求生存與高難度容錯時可考慮 |
| `Daahoud` 3 件 | FC2 `Daahoud` set 頁列出 3 件 bonus 含 `+30 rare drop` 與 `+125 attack ability` | 武僧（Monk） elite 套裝；也可能被物理職拿來評估攻擊能力與 farm 混搭 |
| `Zarth` 3 件 | 來源摘要提到 +15 archery 與 +40 critical hit | 遊俠（Ranger） exceptional 套裝；弓術路線的重要關鍵字 |
| `Falzain` 3 件 | 來源摘要提到 45% magic find，常用 3 件組合觸發 | 天騎士（Templar） exceptional 套裝；以 magic find 混搭為主要賣點 |

### Zamtil's Plenitude：法系抗性腿 {#zamtils-plenitude}

`Zamtil's Plenitude` 是 `Zamtil` elite 套裝的太腿 / 腿部（thigh / legs）部位。FC2 的 `Zamtil` set 頁把它列為 level 65 的 `Abyss Pants`，重點不是單純提高輸出，而是把全屬性耐性、物理耐性、stun / silence resistance、法術相關詞綴與 Enchanter skill enhancement 放在同一個腿部 slot 上。因為高難度地城常先檢查生存與耐性缺口，這件裝備在 FC2 的多個施法或支援職業裝備例中被拿來當抗性腿。

| 出現位置 | FC2 裝備例中的讀法 |
| ------ | ------ |
| 巫師（Wizard） | `Zamtil's Plenitude` 放在太腿 slot；頁面說明該 slot 優先補耐性，雷型可直接採用，火型才視需求比較其他腿部 unique。 |
| 術士（Warlock） | 太腿與靴子用來補耐性；若耐性足夠，才把部分部位換成 magic find 方向。 |
| 牧師（Cleric） | 可用 `Zamtil` 3 件取得 33% magic find 與 13% arcane damage；適合作為 Scion / arcane 方向的 farm 與輸出混搭候選。 |
| 薩滿（Shaman） | FC2 說明部分配裝耐性會偏低，因此以 `Noik's Graft` 和 `Zamtil's Plenitude` 補耐性。 |
| 德魯伊（Druid） / 吟遊詩人（Bard） | 都在裝備例中放入 `Zamtil's Plenitude`，代表它不是 Enchanter 專用思路，而是施法 / 支援職能借用的抗性部位。 |
| 十字軍（Crusader） | 魔法型或抗性吃緊時也可能考慮，屬於「為了通過高壓地城檢查而跨職業借 slot」的案例。 |

<note>
<p>這是 FC2 / 玩家 meta snapshot，不是官方指定的 BiS。公開筆記中可以把 <code>Zamtil's Plenitude</code> 稱為「高難度法系與支援職常見抗性腿候選」，但不要把「最強抗性腿」寫成永久結論；實際仍要比較目前版本 tooltip、roll、角色可穿裝甲類型與隊伍 buff。</p>
</note>

## 拆套與取捨

高階玩家通常不會只問「這套能不能穿滿」，而是問「這個件數 bonus 值不值得犧牲其他部位」。套裝提供穩定、可預期的 bonus；[unique](nevergrind-online-unique-items.md) / legendary 則可能提供更強的單件效果、特殊觸發、技能 roll 或更好的 weapon base。

常見取捨：

- 保留 2 件或 3 件 bonus，把其他部位換成更強的 unique / legendary。
- 放棄全套最後一段 bonus，改用能明顯改善 rotation 的武器或飾品。
- 低等 set 單件若 sockets 很多，可能完全不靠 set bonus，也能靠 rune space 成為後期工具裝。
- magic find 套裝適合 farm，但如果讓輸出或生存掉太多，實際效率可能反而變差。

常見混搭目標可以這樣讀：

| 名稱 | 類型 | 常見取捨 |
| ------ | ------ | ------ |
| `Charlatan's Crest` / Shako | unique head | FC2 unique head 頁列出 `All Talents`、random elemental damage 與 high rare drop；常被拿來替換套裝頭部弱勢 slot |
| `Cryptic Paragon` / Haniwa | unique 1H blunt / focus | FC2 unique 1H blunt 頁列出 `All Talents`、全抗、casting speed、all spell damage 與 undead damage；施法 / 支援職常用來替換不合 rotation 的套裝武器 |
| `Daahoud` 3 件 | monk elite set bonus | 不是只有武僧才值得看；若物理職缺 attack ability 或想兼顧 rare drop，可以把它當成部位成本問題來比較 |
| `Demetrium's Ballista` | low-level set bow | FC2 loot 頁點名為 Normal bow 例外；價值在高 socket roll 與 ranged slot 工具性，不在 level 29 的基礎傷害 |

來源摘要提到一個典型例子：`Yizeren` elite 套裝雖然全套 6 件 bonus 可能提供很高的 `All Talents`，但若套裝兩手武器性能不符合當前施法職流派，玩家可能會拆成 5 件保留 magic find，然後改用 [Cryptic Paragon（Haniwa）](nevergrind-online-cryptic-paragon-haniwa.md) 這類更適合 rotation 的單手武器。

<tip>
<p>評估 set bonus 時，把它當成一個「部位成本」問題：多穿一件套裝，代表你少穿一件其他裝備。只有當 bonus 補到關鍵缺口，或明顯提高 farm 效率時，這個成本才值得。</p>
</tip>

## 收集與整理小提示

- 屬性波動：同名裝備仍可能有不同數值 roll；追求 perfect roll 是後期長線目標。
- Gambling：FC2 gambling 頁提到城鎮右側 `Merchant` 可用 gold gambling；武器 / 防具圖示較固定，飾品圖示較隨機但可 hover 看 `Normal`、`Exceptional`、`Elite`。若追套裝缺件、Shako、Haniwa 或高 socket 工具裝，完整策略可看 [賭博（Gambling）指南](nevergrind-online-gambling.md)。
- 首通獎勵：第一次完成地城或 boss 時，來源摘要提到會有額外 magic find 方向的加成，適合順手追求 set / unique 掉落。
- 鑑定：撿到大量裝備後，可回城找 `Apothecary` 使用 `Identify All` 類功能；完整流程可看 [鑑定（Identification）指南](nevergrind-online-identification.md)，實際按鈕名稱以目前 UI 為準。
- 銀行共享：帳號內共享銀行很適合存其他職業套裝，留給分身角色或未來 build。
- 不要只追滿套：如果 2 件或 3 件 bonus 已經達成目標，剩餘部位可以留給 unique、legendary 或高 socket 裝。

## 和其他裝備系統怎麼接

套裝的價值常和其他系統綁在一起：

- 要先理解綠裝與 set bonus 的定位，可以回到 [物品與戰利品總覽](nevergrind-online-items-loot.md)。
- 若套裝部位有 sockets，接著看 [符文（Runes）指南](nevergrind-online-runes.md)，判斷是否值得投入 rune。
- 若要比較同部位高階裝，可以搭配 [傳奇裝備（Legendary Items）清單筆記](nevergrind-online-legendary-items.md) 查候選裝備。
- 牧師（Cleric）或施法職流派若提到 `Tunso`、`Zamtil`、`Falzain` 這類名字，先確認 bonus 是否真的服務於目前 rotation。
