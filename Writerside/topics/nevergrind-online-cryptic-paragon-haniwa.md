# Nevergrind Online Cryptic Paragon（Haniwa）指南

`Cryptic Paragon`，日系社群常稱 `Haniwa` / `ハニワ` / 哈尼瓦，是 Nevergrind Online 裡法系與支援職會長期追逐的高階 unique 武器。它的價值不只在單一傷害數字，而是同時提供施法節奏、全抗、全法術傷害、對不死生物加成與隨機技能 / 天賦 roll，讓一把武器能服務多種 caster build。

- 版本提醒：公開日文 wiki 與 NotebookLM 摘要的等級和詞綴描述不同；實際數值、等級需求、掉落來源與 gambling 圖示請以目前遊戲內 tooltip 為準

NotebookLM 摘要把 `Cryptic Paragon` 描述為 level 75、1H Blunt / Focus，並強調 `+2 All Talents`、全抗、全法術傷害、不死生物特攻與隨機天賦加成。

日文 wiki 的 `ユニーク武器/片手鈍器` 頁也列出 `Cryptic ParagonHaniwa`，但該頁目前顯示的是 level 70，並列出 attack speed、all resist、casting speed、多種 melee elemental damage、vs humanoid / undead、random skill enhancements、all spell damage、狀態抗性與 maximum mana 等詞綴。

這表示至少有三種可能：

- `All Talents`、`random skill enhancements`、`all skills` 等詞彙在不同資料源裡被混譯或用法不一致。

所以這篇把方向寫清楚，但不把單一數字當成永久定案。真正要交易、賭博或鑲嵌前，請先看當前版本的 tooltip。

## 為什麼它被當成神兵

`Cryptic Paragon` 會被追捧，是因為它同時踩中 caster 後期裝備最想要的幾個方向：

- 提升技能、天賦或隨機技能強化，讓核心 rotation 直接變強。
- 提供 casting haste，讓施法循環更順。
- 提供 all spell damage，對多元素或多技能 build 特別友善。
- 提供 all resist，讓輸出裝不至於完全犧牲生存。
- 提供 undead damage，對 `Riven Grotto`、crypt 系區域與 Cleric / Paladin 類 undead farm 很有價值。
- 作為 1H weapon，可以和 off-hand、shield 或雙持配置形成不同玩法。

它不是單一職業專武，而是一把「很多法系都能找到用途」的高階泛用裝。也因此，真正稀有的不是掉到一把 Haniwa，而是掉到一把剛好對你的職業、技能和裝備配置都對的 Haniwa。

## 職業應用

| 職業 / 流派 | 追求重點 |
| ------ | ------ |
| `Cleric` | `Condemnation`、`Deliverance`、`Sacred Revelation`、undead damage、casting haste |
| `Druid` | 如果隨機技能 roll 命中 `Tornado` 或核心元素技能，價值會大幅提高 |
| `Wizard` | 追 lightning / fire / ice 核心技能、casting haste、all spell damage；高階配置可比較雙持或切武器 |
| `Enchanter` | 若 roll 到 `Superior Enthrall` 或核心控制 / haste 相關技能，會直接改善控制與支援節奏 |
| `Shaman` | 補基礎火力、DoT / spell damage、資源與技能等級；雙持玩法要看版本與 proficiency |
| `Bard` | 看歌曲、support skill、magic find 與 all talents / all skills 是否支援目前定位 |

對 [Scion 牧師輸出](nevergrind-online-cleric-scion-dps.md) 來說，Haniwa 的價值在於同時補輸出、節奏與 undead farm；對 [牧師刷區域](nevergrind-online-cleric-farming-zones.md) 來說，它則是把 undead 特攻路線推到更高效率的候選裝備。

## 取得方式

NotebookLM 摘要提到兩個方向：地城掉落與 [gambling](nevergrind-online-gambling.md)。公開 Fandom Loot 頁能確認 merchants 會依角色等級販售裝備並定期 restock，但未明確列出 Haniwa gambling 規則；因此 gambling 細節先當社群實測資訊處理。

整理如下：

| 方式 | 說明 |
| ------ | ------ |
| 地城 / boss 掉落 | 依 monster type、magic find、首通、難度、任務掉落傾向與 RNG 決定 |
| Gambling | NotebookLM 摘要建議鎖定 1H Blunt 圖示，嘗試賭出 unique |
| 交易 / 社群 | 若市場存在合適 roll，有時比自己硬刷更有效率 |

NotebookLM 摘要提到「132 次出現 5 個 unique，約 3.78%」的測試數據。這個數字可以當 gambling 風險感參考，但不要視為固定機率；版本、角色等級、部位池、權重與樣本數都可能讓實際體感不同。

## 怎麼判斷一把 Haniwa 值不值得留

拿到 `Cryptic Paragon` 後，先不要只因為名字興奮，照這個順序看：

1. 角色等級與職業能不能裝。
2. `All Talents` / `All Skills` / random skill roll 是否命中你的 build。
3. casting haste 是否能讓 rotation 接近下一個 breakpoint。
4. all spell damage 是否支援你的主要傷害型態。
5. all resist 是否補到高難度所需門檻。
6. undead damage 是否符合你的 farm 路線。
7. sockets / rune plan 是否值得投入高階符文。
8. 和目前武器相比，是全面提升，還是只多了 magic find / 特定技能。

## 配裝與符文方向

Haniwa 這類高價值 weapon 通常值得認真規劃 [符文](nevergrind-online-runes.md)，但前提是 roll 夠好。

常見方向：

- caster：優先看 `Mael` 類 enemy resistance reduction、casting haste、crit、all spell damage。
- Cleric：補 `Condemnation` / `Deliverance` 相關技能、undead damage、spirit / mana sustain。
- Druid / Wizard：依主要元素選擇傷害、resistance reduction 或 casting speed。
- 支援職：如果主職責是 buff / control，技能等級與資源穩定性可能比純傷害更重要。
- farm 裝：已有足夠輸出與生存時，再考慮 magic find 類方向。

<warning>
<p>不要把普通 roll 的 Haniwa 直接塞上最貴 rune。高階 rune 的機會成本很高，先確認這把武器的隨機技能與當前 build 真正契合。</p>
</warning>

## 常見誤區

- 看到 `Cryptic Paragon` 就當作所有 caster 的無條件 BiS。
- 只追 `+2 All Talents`，忽略隨機技能 roll、haste、抗性與資源。
- 為了 magic find 犧牲到 Hell 難度生存。
- 把日文 wiki 舊數值、NotebookLM 摘要與當前遊戲 tooltip 混成同一版。
- 沒確認 1H Blunt / focus 類裝備是否符合自己職業 proficiency。
- 先鑲嵌高價 rune，後來才發現 roll 不適合。
