# Nevergrind Online 獨特裝備（Unique Items）指南

Nevergrind Online 的紫色 `Unique` 裝備代表獨特物品：詞綴種類大致固定，但具體數值會在範圍內浮動。這讓 unique 比黃色 rare 更可預期，也比綠色 set 更適合補單一部位或關鍵技能，是中後期 build 優化的核心目標之一。

- 檢視日期：`2026-05-05`
- 分類：[Nevergrind Online 物品與戰利品總覽](nevergrind-online-items-loot.md)
- 資料來源：來源摘要、Swiftyhorn / 高階玩家指南相關來源摘要、FC2 / atelier3 unique、loot 與 selected unique 頁
- 版本提醒：unique 名稱、等級、詞綴、gambling 規則與掉落來源可能隨版本不同而調整；實際判斷請以目前遊戲內 tooltip 為準

<tldr>
<p>Unique 的價值來自「固定 mods + 隨機數值 roll」：方向穩定，但仍要追好數值。</p>
<p>後期主要追 Elite 階級 unique，因為底材攻擊、防禦與需求會影響最終上限。</p>
<p>Unique 常拿來替換套裝弱勢部位，或提供套裝給不了的技能、天賦、特殊效果與 weapon base。</p>
</tldr>

## 獨特（Unique）在稀有度體系中的位置 {#unique-rarity-position}

紫色 `Unique` 和黃色 `Rare` 最大的差異在穩定性。Rare 的詞綴組合高度隨機，神 roll 時可能非常強；unique 則通常有一組固定 mods，玩家主要比較的是數值 roll、隨機技能 / 天賦是否命中，以及這件裝備是否適合目前 build。

| 類型 | 詞綴結構 | 判斷方式 |
| ------ | ------ | ------ |
| `Rare` | 2 到 6 個隨機 mods | 看詞綴組合是否剛好補到需求；延伸看 [稀有裝備（Rare Items）指南](nevergrind-online-rare-items.md) |
| `Unique` | 固定 mods，數值浮動 | 看固定效果是否支援 build，再追好 roll |
| `Set` | 固定 mods，加上多件 bonus | 看要穿幾件才值得犧牲其他部位 |

來源摘要提到，特定 unique 可能有 6 個以上固定 mods；即使 mods 種類固定，像 health leech、magic find、all talents、resists、skill roll 這類數值仍可能有高低差。也因此，掉到一件 unique 只是開始，掉到符合職業、技能與數值需求的版本才是真正目標。

## 品質階級

Unique 也會受到裝備品質階級影響。`Normal`、`Exceptional`、`Elite` 不只是名稱，而會影響底材基礎性能、等級需求與後期價值。

| Quality | 常見階段 | 看裝重點 |
| ------ | ------ | ------ |
| `Normal` | 前期與過渡 | 可用來補技能或抗性，但通常不要過度投入高階 rune |
| `Exceptional` | 中期主力 | 若固定效果很好，可以撐很久，特別是飾品或特殊功能裝 |
| `Elite` | 後期目標 | 主要追求對應 build 的高 roll、技能命中、socket / rune plan 與交易價值 |

後期玩家通常會優先追 elite unique，因為同樣是強效果，底材階級會讓基礎 damage、armor 或等級需求拉開很大差距。

## 值得記的獨特（Unique）關鍵字 {#unique-keywords}

這些名字來自來源摘要與既有 Nevergrind 筆記，適合當 farm、trade、gambling 或裝備比較時的關鍵字。實際等級與詞綴仍請以遊戲內 tooltip 為準。

| 名稱 | 重點 | 常見用途 |
| ------ | ------ | ------ |
| [Cryptic Paragon（Haniwa）](nevergrind-online-cryptic-paragon-haniwa.md) | 來源摘要描述為高階 1H blunt；看 `All Talents`、全抗、全法術傷害、undead damage 與隨機技能 / 天賦 roll | 施法職 / 支援 / 牧師（Cleric）類流派的長期追求目標 |
| `Charlatan's Crest` | 來源摘要提到 all talents、屬性傷害與 high magic find；也常被稱為 shako 類頭盔目標 | 施法職、牧師（Cleric）或 farm 裝常見關鍵字 |
| `Marshal Gauntlets Iniquity` | 來源摘要描述為高階 plate gloves，對 plate 職很有價值；FC2 loot 頁也把 rare gloves 視為刷到目標 tree roll 前的過渡 | 戰士（Warrior）、牧師（Cleric）、吟遊詩人（Bard）、暗影騎士（Shadow Knight）等 plate 職的 talent / 裝備候選 |
| `Stormcaller Bow` | FC2 loot 頁點名的 Normal bow 例外；若 roll 到高 sockets，可能因 ranged slot 與 rune space 留到後期 | 物理職或需要 ranged slot 工具裝的角色，先比較 socket count、talents 與目前 build 需求再出售 |
| `Chaos Mirror` | 來源摘要提到 tree talents 與 specific talent 方向 | 特定技能強化或 build breakpoint，比泛用戰力更偏專精用途 |

FC2 另有一頁 [各職の厳選ユニーク](https://atelier3.web.fc2.com/ngo/selectlist2.html)，專門列出「其他職業掉落時也值得先看一眼」的 Unique / 特殊單件。這類清單不適合直接當 BiS 表，但很適合拿來建立銀行保留規則：看到名字後先檢查 talent、specific skill、DPS、socket count、rare drop、主屬性或抗性是否命中。整理後的本地速查可看 [FC2 各職嚴選 Unique 裝備速查](nevergrind-online-fc2-selected-unique-items.md)。

<note>
<p>這些名稱適合用來建立搜尋與比較方向，不代表任一版本的固定 BiS。真正交易、gambling 或投入 rune 前，先確認當前版本的等級、部位、proficiency、roll range 與可裝備職業。</p>
</note>

## 和套裝的取捨

Unique 常被拿來替換套裝中的弱勢部位。這不是因為 set bonus 不強，而是高階配裝要同時計算「多穿一件套裝」和「少穿一件強 unique」的機會成本。

常見情境：

- 套裝武器或特定部位太弱，改用更強 unique 來改善 DPS、casting flow 或 skill roll。
- 保留 2 件、3 件或 5 件 set bonus，把剩餘部位交給 unique / legendary 補關鍵缺口。
- 單件 unique 提供套裝沒有的特殊效果、特定 talent、enemy resistance reduction 或 magic find。
- 某些 build 不需要全套 bonus，反而需要一件能讓 rotation 成立的 unique。

例如來源摘要提到 `Yizeren` elite 套裝的兩手武器若不適合施法職流派，玩家可能拆套保留部分 magic find，再改用 Haniwa 類 unique weapon 來提高實戰輸出與技能契合度。這種取捨也是 [套裝（Set Items）指南](nevergrind-online-set-items.md) 裡強調的核心觀念。

## 取得方式

Unique 的取得通常是一個長期坑，因為你不只要掉到名字，還要掉到好 roll。

- Gambling：來源摘要提到金幣充裕後，可以在城鎮商人處鎖定特定圖示 gambling。飾品類，例如 ring、amulet、charm，常被視為值得投入的目標之一；完整策略可看 [賭博（Gambling）指南](nevergrind-online-gambling.md)。
- 怪物掉落：來源摘要提到 unique monsters 會掉落多件藍色以上裝備，是追 unique 的重要來源。
- Magic find：magic find、首通、boss、隊伍人數、難度、monster con 與 quest preferred item types 都可能影響你看到高稀有度裝備的機會。
- 交易與共享銀行：好 unique 不一定只服務當前角色；若 roll 合適，留給分身或交易也很合理。

<tip>
<p>追 unique 時，先定義「我要補什麼」：是 all talents、特定技能、magic find、抗性、weapon base、還是特殊效果。沒有目標地追紫裝，很容易花很多 gold 卻不知道哪一件真的變強。</p>
</tip>

## 判斷一件獨特（Unique）裝值不值得留 {#unique-keep-checklist}

1. 部位是否正在卡 build。
2. Quality tier 是否夠高，或它是否因特殊效果值得跨階使用。
3. 固定 mods 是否真的支援目前職業、技能與隊伍定位。
4. 數值 roll 是否接近你需要的門檻。
5. 隨機技能 / 天賦是否命中核心 rotation。
6. 是否比同部位 set bonus、rare 神 roll 或 legendary 候選更有價值。
7. 是否值得投入 rune、升級、交易成本或銀行空間。
