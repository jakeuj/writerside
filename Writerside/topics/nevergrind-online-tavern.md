# Nevergrind Online Tavern 酒館指南

Nevergrind Online 的 `Tavern` 酒館是 Edenburg 裡把玩家送進冒險的核心入口：想接任務、進地城、確認任務掉落偏好、查看排行榜或讀新手 tips，都會回到這裡。出發前先在酒館確認任務目標、隊伍 ready 狀態與 farm 方向，比只盯著下一場地城更能穩定推進角色成長。

## Tavern 在 Edenburg 的角色

Edenburg 是每場地城前後的整理台，而 `Tavern` 則是從休整狀態切換到冒險狀態的門口。來源摘要把酒館描述為位於 Edenburg 左側的設施，主要負責三件事：給任務、送玩家前往地城、提供排行榜與 tips 之類的資訊入口。

可以先把酒館理解成這幾個功能的集合：

| 功能 | 用途 | 為什麼重要 |
| ------ | ------ | ------ |
| `Missions` | 選任務、進地城、挑戰 boss | 決定這一場要打哪個環境、哪個目標與哪種掉落期待 |
| 掉落偏好 | 來源摘要提到每項任務有兩種更容易掉落的裝備類型 | farm 特定部位時，不只看怪物，也要看任務偏好 |
| `Leaderboards` | 查看角色進度、ladder 排名或競爭狀態 | 適合追等級、賽季紀錄或高階競爭目標 |
| `Tips` | 讀基礎教學、快捷鍵與聊天指令提示 | 新手先看這裡，可以少走很多冤枉路 |

## 任務與冒險入口

來源摘要提到，玩家需要透過酒館選擇 `Missions` 才能進入地城；當時整理的資料也指出遊戲提供超過 100 種任務，每個任務會把玩家送往特定環境，挑戰不同 boss 或任務目標。實際數量與分類可能隨版本更新，出發前仍以遊戲內列表為準。

選任務時不要只看等級。比較實用的判斷順序是：

1. 這個任務等級是否符合角色與隊伍強度。
2. 這個環境的怪物類型是否適合你的職業，例如 牧師 (Cleric) 偏好 undead / demon 場景時，可以搭配 [牧師刷區域指南](nevergrind-online-cleric-farming-zones.md) 評估。
3. 任務偏好的掉落裝備類型，是否符合目前想 farm 的部位。
4. boss 與地城危險度，是否值得為首通、magic find 或掉落期待承擔。
5. 隊伍是否已經補給、鑑定、修整與 ready。

<tip>
<p>如果你在追特定部位，不要只記 boss 名稱。任務本身的 preferred item types 可能讓某些路線更適合 farm；這也是 Tavern 從「傳送選單」變成刷寶策略中心的原因。</p>
</tip>

## Leaderboards 與 Ladder

酒館也是查看競爭狀態的地方。來源摘要提到，`Leaderboards` 讓玩家確認彼此進度；在 `Ladder` 模式下，排行榜更像是賽季目標的一部分，玩家會追求等級、進度或紀錄排名。

可以這樣看排行榜：

- 休閒玩家：確認目前活躍職業、常見等級段與伺服器節奏。
- 練新角色：觀察同職業玩家的進度，理解哪些 build 或路線可能熱門。
- 追賽季：把 ladder 排名當作長期目標，注意賽季結束後角色與紀錄如何轉入 eternal 類環境。
- 研究 meta：從排行榜上的職業與等級分布，反推當前版本哪些職業或 farm 路線比較有效率。

<note>
<p>來源摘要提到賽季可能約每三個月更迭，ladder 角色成果最後會轉入 eternal 伺服器。這類營運規則很容易因版本與賽季調整，正式衝榜前請先看遊戲內公告與目前 leaderboard 說明。</p>
</note>

## Tips 分頁與新手資訊

酒館不只是任務入口，也像一個遊戲內導師。來源摘要特別建議新手打開上方的 `Tips` 分頁，因為這裡會整理基礎教學、快捷鍵、聊天指令與一些進階提示。

新手可以優先找這些資訊：

- 介面操作與快捷鍵。
- 組隊、邀請、隊伍聊天或 party help 相關指令。
- 地城出發、ready、隊長帶隊與戰鬥節奏提示。
- 背包、商店、補給、鑑定或回城整理的注意事項。
- 其他遊戲內 `help` 或 tips 入口。

如果你正在學社交與組隊流程，也可以搭配 [熱鍵與聊天指令速查](nevergrind-online-hotkeys-commands.md) 和主指南裡的 [操作與指令速查](nevergrind-online-guide.md#ngo-chat-command-quick-reference) 一起看。

## 組隊出發流程

酒館是隊伍真正進地城前的最後確認點。來源摘要提到，隊伍選定任務後，隊員需要處於 ready 狀態；常見 UI 表示可能是頭像旁的綠色小劍，接著由隊長負責帶隊進入地城，也就是玩家俗稱的 driving / 開車。

建議出發前照這個順序檢查：

1. 隊長在 `Tavern` 選好任務與難度。
2. 每位隊員確認藥水、卷軸、背包空間與裝備狀態。
3. 需要 farm 特定部位時，先確認任務偏好是否符合目標。
4. 隊員切到 ready 狀態。
5. 隊長帶隊進入地城。
6. 進場後先等 tank 或指定開怪者建立節奏，再開始輸出、控制或治療。

這套流程聽起來很基本，但在 NGO 這類短場次地城 RPG 裡很重要。少一次忘記補給、少一次隊伍還沒 ready 就出發，整體 farm 節奏會順很多。

## 什麼時候該回酒館

酒館不是只在登入後開一次。只要你的目標從「打一場」變成「有效率地刷」，就會一直回到它。

常見回酒館的時機：

- 想換任務或改 farm 部位。
- 想看某條路線的 boss / mission 是否適合目前 build。
- 隊伍成員更換，需要重新確認目標。
- 想看 leaderboard，確認 ladder 進度或職業分布。
- 新手想查 tips、help、party 指令或操作提示。

總結來說，`Tavern` 是 NGO 把「目標、資訊、競爭與社群出發」綁在一起的地方。把它用好，地城不會只是隨便選一場進去，而會變成有方向的練等、刷寶與隊伍推進。
