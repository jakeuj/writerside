# Nevergrind Online 賭博（Gambling）指南

Nevergrind Online 的 `Gambling` 賭博是後期金幣出口：技能、背包與基本補給穩定後，玩家可以在 Edenburg（伊登堡）的 `Merchant` 用 gold 鎖定特定部位，嘗試取得 unique、set、legendary 或其他高價值裝備。它不是新手的第一筆投資，而是「已經知道自己要追哪個部位」之後，用多餘金幣換裝備機會的長線玩法。

## 賭博（Gambling）在經濟循環中的位置

Gambling 的本質是把 gold 換成定向抽裝機會。它和一般地城掉落不同：你可以選擇某個圖示或部位，但仍要承擔稀有度、品質、roll、隨機技能與實際可用性的 RNG。若還在煩惱「gold 應該先升技能、背包，還是拿來賭」，先看 [金錢效率與賣裝策略](nevergrind-online-money-efficiency.md)。

比較適合開始 gambling 的時機：

- 常用技能已經升起來，角色能穩定完成目標地城。
- [商人（Merchant）](nevergrind-online-merchant.md) 的背包擴張已經做到不會常常滿包。
- 補給、鑑定與修整不會吃光全部 gold。
- 你已經知道要追哪個部位，例如 amulet、ring、elite head、1H blunt 或特定 armor slot。
- 你能接受幾十次甚至幾百次都沒有目標裝備。

不建議太早投入的情況：

- 核心 rotation 還沒成形，技能升級更有價值。
- 角色缺抗性、生存或補給，地城本身還不穩。
- 你只是看到「可能出 legendary」就想梭哈，卻沒有明確部位目標。

## 地點與重置

來源摘要把 gambling 位置整理在 Edenburg（伊登堡）右側的 `Merchant`。這和一般商店買賣、背包擴張同屬商店系統，但 gambling 應該視為更高風險的後期金幣用途。

來源摘要提到的操作觀念：

- 武器與防具：圖示相對固定，可以依想追的裝備類型選圖示。
- 飾品類：amulet、ring、charm 的圖示可能較隨機，需靠 hover tooltip 看品質。
- 品質階級：hover 後確認 `Normal`、`Exceptional`、`Elite`，再決定是否購買。
- 重置條件：來源摘要提到進入地城或 relog 可能重置品項；實際行為請以目前版本為準。

<warning>
<p>Gambling 的重置、legendary 加成與稀有度權重很容易被 patch 調整。本文把它寫成來源摘要中的策略，不把任何機率或保證規則當成永久定案。</p>
</warning>

## 品質階級怎麼看

`Normal`、`Exceptional`、`Elite` 不是裝飾文字。它們會影響底材等級、可能裝備池與後期價值。賭博時尤其要避免買錯品質，因為同樣是「頭盔」或「鈍器」，不同 quality 可能對應完全不同的目標池。

| 品質 | 常見用途 | Gambling 判斷 |
| ------ | ------ | ------ |
| `Normal` | 前期過渡、低等特殊件 | 除非追特定低等 set / unique，否則後期通常不是主目標 |
| `Exceptional` | 中期關鍵部位 | 可追某些高性價比 set / unique，但要確認部位池 |
| `Elite` | 後期目標 | 追 Haniwa、Shako、elite 飾品與高階 legendary 候選時最常見 |

來源摘要特別提到，Exceptional / Elite 的 amulet、ring、charm 在 gambling 中可能沒有黃色 rare 結果，因此被視為高 CP 值目標。這條規則非常重要，但也最需要用目前版本 tooltip 或社群測試確認；若版本改過，策略價值會一起改變。

## 優先目標

下面的目標來自來源摘要與既有筆記，適合當 gambling 清單，不代表任何版本的固定 BiS。

| 目標 | 對應方向 | 為什麼值得看 |
| ------ | ------ | ------ |
| Elite amulet / ring / charm | 精英飾品 | 來源摘要認為是最穩定的 unique 以上候選；也容易跨職業使用 |
| `Shako` / elite head | `Charlatan's Crest` | 來源摘要提到 `All Talents +2`、屬性傷害與高 magic find，是 farm 裝常見追求 |
| `Haniwa` / elite 1H blunt / focus | [Cryptic Paragon](nevergrind-online-cryptic-paragon-haniwa.md) | 施法職 / 支援 / 牧師（Cleric）類流派常追的 `All Talents`、全抗、全法術傷與隨機技能武器 |
| Elite staff 類 | `Firmament Staff of the Crystal Sea`、`Thoth Mindlink Staff` 等 | 來源摘要提到可能作為強力 legendary 追求；機率與池子需以當前版本確認 |
| Boots | `Trek of Glory` | 來源摘要提到 magic find 與經驗加成，適合 farm 取向 |
| Set bow | `Demetrium's Ballista` | 低等 set bow 但 sockets 很多，可能作為 rune space 工具裝 |
| Elite armor / gloves | `Courtier's Trunk`、`Marshal Gauntlets Iniquity` | 特定部位可能提供 talents、防禦或 build breakpoint |

<tip>
<p>剛開始 gambling 時，先追 elite 飾品通常比直接追單一神裝溫和。飾品更容易跨職業、跨 build 使用，也比較不會因 weapon proficiency 或 armor tier 卡住。</p>
</tip>

## 傳奇（Legendary）與機率心態

來源摘要提到 gambling 可能出現 legendary，但通常需要大量嘗試；也提到近期更新似乎讓 legendary 掉落加成能作用於 gambling。這種說法在策略上可以記住，但不適合寫成固定保證。

比較穩的心態：

- 把 legendary 當超低機率額外驚喜，不要當每輪 gambling 的基本期待。
- 先追「部位正確且可用」的 unique / set，再期待 legendary。
- 若 patch note 或遊戲內 UI 顯示 legendary bonus 會影響 gambling，再調整投入順序。
- 記錄自己的嘗試次數與結果，避免只靠體感判斷機率。

如果想查 legendary 候選名稱、quality、slot 與 proficiency，可搭配 [傳奇裝備（Legendary Items）清單筆記](nevergrind-online-legendary-items.md)。

## 建議流程

1. 先決定目標部位，例如 elite amulet、elite ring、elite head、elite 1H blunt。
2. 到 [商人（Merchant）](nevergrind-online-merchant.md) 找對應 gambling 圖示。
3. 對飾品類先 hover，確認是 `Normal`、`Exceptional` 還是 `Elite`。
4. 購買後用 [鑑定（Identification）指南](nevergrind-online-identification.md) 的流程確認 mods、roll 與可用性。
5. 自用候選留下，分身或交易候選放銀行，無用裝備賣回商店。
6. 若來源摘要中的重置條件符合目前版本，可透過進地城或 relog 刷新品項。
7. 定期停下來檢查 gold 是否該轉回技能、背包、補給、runes 或 crafting。

## 常見錯誤

- 沒有先升背包與技能，就把早期 gold 全丟進 gambling。
- 沒有 hover 飾品品質，買到不符合目標池的 item。
- 看到 `Elite` 就以為一定是 BiS，忽略職業 proficiency、roll 與 build 需求。
- 追 [Cryptic Paragon（Haniwa）](nevergrind-online-cryptic-paragon-haniwa.md) 卻不檢查隨機技能是否命中。
- 只看 unique / legendary 名字，不看是否值得投入 [符文](nevergrind-online-runes.md)。
- 把單一玩家的小樣本機率當成固定掉率。

Gambling 好玩的地方，是它讓多餘 gold 變成有方向的裝備追求；危險也在這裡。最好的策略不是一直按，而是知道自己在追什麼、為什麼追、什麼時候該停。
