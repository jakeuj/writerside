Jetbrains IDE 內建 stash 和 shelve

## 結論

我不知道差在哪

* stash
  + 會把改的檔案還原回未變更之前的檔案，同時將差異的部分暫存起來
  + 之後可以再 unstash，把檔案在從原始版本變成剛剛修改後的內容
* shelve
  + 會把改的檔案還原回未變更之前的檔案，同時將差異的部分暫存起來
  + 之後可以再 unshelve，把檔案在從原始版本變成剛剛修改後的內容
