# Nevergrind Online Merchant 商店指南

Nevergrind Online 的 `Merchant` 商店是 Edenburg 裡的後勤中心：它不只賣部分輕型裝備與隨機商品，更重要的是提供背包空間擴張。初期金幣常在技能、補給與背包之間拉扯；如果你常在地城裡滿包，就該把 Merchant 的 inventory upgrade 當成提升刷寶效率的長期投資。

- 檢視日期：`2026-05-03`
- 分類：[Nevergrind Online（絕不刷怪）遊戲指南](nevergrind-online-guide.md)
- 資料來源：NotebookLM 摘要、Fandom / 公開 wiki 與來源資料整理
- 版本提醒：背包上限、升級費用、商人名稱、restock 週期、快捷鍵與販售品類可能因版本更新或語系不同而調整；實際操作請以目前遊戲內 UI 與 tooltip 為準

<tldr>
<p>Merchant 的核心價值是 inventory upgrade：背包越大，越能帶補給，也越能把高價戰利品帶回城。</p>
<p>來源摘要提到背包可從 8 格逐步擴到 40 格，且每個角色都要各自支付升級費用。</p>
<p>Merchant 也負責 leather armor、隨機裝備與定期 restock，是刷寶之外的補裝來源。</p>
</tldr>

## Merchant 在 Edenburg 的定位

Edenburg 裡每個商業設施都有自己的重點。`Blacksmith` 偏重甲、重武器與後期強化，[Apothecary](nevergrind-online-apothecary.md) 偏補給、鑑定與法系需求，`Merchant` 則把重點放在背包、一般商品、輕型防具與隨機庫存。

| 設施 | 裝備 / 功能重點 | 使用情境 |
| ------ | ------ | ------ |
| `Merchant` | 背包擴張、`Leather Armor`、一般商品、隨機庫存 | 初期提升刷寶續航、補皮甲或檢查商店刷新 |
| `Blacksmith` | `Plate`、`Mail`、重型武器、runes / crafting | 重裝職業補裝、socketed items 後期客製化 |
| [Apothecary](nevergrind-online-apothecary.md) | 藥水、鑑定卷軸、`Identify All`、偏法系需求 | 地城後補給、批量鑑定與 caster 方向整理 |

如果 `Tavern` 是出發入口，Merchant 就是讓下一場地城更有效率的後勤站。背包太小時，你不是只少撿幾件裝備，而是會被迫更常回城、更早丟掉未鑑定物，甚至少帶藥水。

## 背包擴張與 Twiddles

來源摘要把背包擴張視為 Merchant 最重要的服務。玩家初始背包空間很小，資料中整理為 8 格；透過名為 `Twiddles` 的商人支付金幣升級後，可逐步擴展，摘要提到上限可到 40 格。

背包升級的重點：

- 費用會逐次增加，不是一次買斷。
- 背包擴張是角色獨立的，每個角色都要各自升級。
- 背包越大，越能帶回未鑑定裝備、高價底材、藥水與任務途中掉落。
- 背包升級通常和技能學習、補給、鑑定成本競爭同一筆金幣。

<note>
<p>來源摘要提到背包升級入口通常在 Merchant 視窗左下角，且商人名稱為 <code>Twiddles</code>。UI 位置與名稱可能因版本或語系不同，找不到時先看目前商店視窗提示。</p>
</note>

## 什麼時候該先升背包

背包不是純舒適性功能，它會直接改變地城收益。若你常在 boss 前就滿包，或不得不丟掉 yellow / blue 未鑑定裝備，背包升級的收益會很快浮現。

優先升背包的情境：

1. 每場地城都會滿包，導致你無法帶回高價裝備。
2. 你開始想保留 set、unique、抗性裝或分身裝。
3. 你需要多帶藥水、卷軸或資源補給。
4. 你正在 farm 特定部位，想提高每趟可帶回的候選數量。
5. 技能主循環已經能順利打怪，短期傷害提升不如多帶戰利品重要。

可以暫緩的情況：

- 核心技能還沒升起來，導致地城本身打不順。
- 你還在非常前期，裝備汰換速度極快。
- 目前路線短、回城頻繁，背包壓力還不大。

## Leather Armor 與隨機商品

Merchant 也是 `Leather Armor` 的主要供應點。對 `Druid`、`Monk`、`Templar`、`Wizard`、`Warlock` 這類最高穿到 leather 的職業來說，商店刷新有時能提供不錯的過渡裝。

看 Merchant 裝備時，優先確認：

- 等級是否貼近目前角色。
- 是否符合職業 armor tier 或 weapon proficiency。
- 詞綴是否補到抗性、生命、資源、主屬性、haste 或 talents。
- 價格是否合理，還是把金幣留給技能、背包與補給更划算。

來源摘要與公開 loot 資料都提醒，merchants 會依角色等級提供裝備，且會定期 restock。摘要提到庫存約每 1 小時的角色在線時間刷新；實際週期仍以目前版本為準。這讓商店巡視變成刷寶之外的小型抽卡：多半只是過渡品，但偶爾會遇到剛好補洞的好裝。

## 買賣與快捷操作

Merchant 是清理低價值戰利品、換取金幣和補齊一般需求的主要窗口。來源摘要提到 `Ctrl + Left Click` 可在商店窗口快速買賣；既有社群 tips 也曾提到 `Shift + Left Click`。快捷鍵很容易因版本、平台或設定不同而變動，因此最穩的做法是以目前 UI / keybind 提示與 [熱鍵與聊天指令速查](nevergrind-online-hotkeys-commands.md) 為準。

## Gambling 與後期金幣出口

當技能、補給與背包擴張都穩定後，Merchant 也會變成後期 gambling 的入口。這時 gold 的用途不再只是買賣與補裝，而是用來鎖定特定圖示或部位，嘗試取得 unique、set 或 legendary 候選。

如果你剛開始碰 gambling，先看 [Gambling 賭博指南](nevergrind-online-gambling.md)，再決定要追 elite 飾品、Haniwa、Shako 或其他特定部位。

常見買賣流程：

1. 回城後先鑑定高價候選，尤其是 [Identification 鑑定指南](nevergrind-online-identification.md) 提到的 talents、高抗性與高價部位。
2. 自用、分身或交易候選先放一邊。
3. 低價、低等、詞綴不合 build 的裝備賣掉。
4. 檢查 Merchant 是否有背包升級、皮甲替代品或剛好補短板的商品。
5. 金幣先分配給技能、背包、補給，再考慮 [Gambling](nevergrind-online-gambling.md) 或後期投入。

<tip>
<p>如果你常常「背包滿到不想撿」，那不是 RNG 對不起你，而是後勤該升級了。Merchant 的背包擴張會讓每趟地城更像完整收益，而不是半路清倉。</p>
</tip>

## 和其他設施搭配

Merchant 的價值通常和整個 Edenburg 流程一起看：

1. 從 [Tavern 酒館](nevergrind-online-tavern.md) 選任務與 farm 目標。
2. 地城後到 [Apothecary](nevergrind-online-apothecary.md) 鑑定與補給。
3. 重甲、重武器與 socketed items 交給 [Blacksmith 鍛造鋪](nevergrind-online-blacksmith.md) 評估。
4. 到 Merchant 賣掉低價戰利品、升背包、檢查 leather 或隨機裝備刷新。
5. 把剩下的高價裝、set 件、分身裝與抗性裝放進 bank。

總結來說，Merchant 是 NGO 的後勤槓桿。它賣的裝備未必永遠最強，但背包擴張會讓所有職業的刷寶效率變好；懂得把金幣分配在技能、背包與補給之間，是前期經營最重要的節奏感。
