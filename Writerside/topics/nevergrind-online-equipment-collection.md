# Nevergrind Online 裝備收集路線

Nevergrind Online 的裝備收集不是只把顏色往上換，而是把「稀有度、品質階級、詞綴 roll、掉落來源、金幣投資、銀行保存與鐵匠強化」串成長期路線。新手先學會保留能補洞的藍黃裝，中期開始追 unique、set 與高價部位，後期再把 sockets、runes、craft base、Ethereal / Indestructible 放進同一個收集計畫。

- 檢視日期：`2026-05-05`
- 分類：[Nevergrind Online 物品與戰利品總覽](nevergrind-online-items-loot.md)
- 延伸閱讀：[FC2 各職代表技能速查](nevergrind-online-fc2-signature-skills.md)、[FC2 各職嚴選 Unique 裝備速查](nevergrind-online-fc2-selected-unique-items.md)、[金錢效率與賣裝策略](nevergrind-online-money-efficiency.md)、[賭博（Gambling）指南](nevergrind-online-gambling.md)、[鐵匠鋪製作與配方（Blacksmith Crafting / Recipe）深度筆記](nevergrind-online-blacksmith-crafting-recipe-research.md)
- 資料來源：來源摘要、既有 Nevergrind 筆記、Fandom Loot / Items / Races、Nevergrind Wiki、FC2 / atelier3 攻略 DB 與 SteamDB patch notes
- 版本提醒：稀有度顏色、legendary / mythical 分類、掉落保證、商店 restock、gambling 池與 craft 規則可能隨版本變動；最後判斷請以目前遊戲內 tooltip、酒館（Tavern）任務資訊與 鐵匠鋪（Blacksmith） / 商人（Merchant） UI 為準

<tldr>
<p>收集裝備時先問「這件補哪個 build 缺口」，再問「它是什麼顏色」。</p>
<p>Magic / Rare 不是純過渡；高抗性藍裝、好詞綴黃裝、rare amulet / gloves 都可能撐很久。</p>
<p>後期收藏重點會轉向 unique / set / legendary、high socket 工具裝、Ethereal + Indestructible、以及可做 Mythical / Rune Words 的好 base。</p>
</tldr>

## 先建立收集層級

收集裝備可以用三層來看：

| 層級 | 你在收什麼 | 代表問題 |
| ------ | ------ | ------ |
| 生存與過渡 | 抗性、HP、mana / spirit、armor、主屬性、基本 weapon damage | 這件能不能讓我下一場地城更穩 |
| Build 成形 | `All Talents`、技能 / talent tree 加成、haste、crit、attack、特定傷害類型 | 這件是否讓 rotation 或隊伍定位更完整 |
| 終局收藏 | 高 roll unique、set bonus、legendary、socketed base、Ethereal / Indestructible、Mythical craft | 這件是否值得放銀行、交易或投入 rune / crafting |

這三層會重疊。一件藍色高抗性裝可能只是過渡，也可能是讓你打過某個 boss 的關鍵；一件紫色 unique 如果 random skill 沒命中，也可能只是高價賣店或交易品。

## 稀有度與品質階級

完整稀有度可回到 [物品與戰利品總覽](nevergrind-online-items-loot.md#item-system-loop)，這裡只整理收集時的判斷方式。

| 顏色 / 類型 | 收集意義 | 典型保留理由 |
| ------ | ------ | ------ |
| 藍色 `Magic` | 少量隨機詞綴，但單一詞綴可能很高 | 高 resistance、高資源、特定短板補強 |
| 黃色 `Rare` | 2 到 6 個隨機詞綴，組合上限高 | rare amulet、rare gloves、主屬性 + talents + 抗性 |
| 紫色 `Unique` | 固定 mods，數值與部分 roll 仍會浮動 | Haniwa、Shako、Chaos Mirror 類固定方向裝 |
| 綠色 `Set` | 固定 mods + 多件 set bonus | 職業套裝、MF 組合、特定 2 件 / 3 件 bonus |
| 橙色 `Legendary` | 來源摘要提到的極稀有高階掉落 | 強特殊效果、技能強化或改變玩法的裝備 |
| `Mythical` / Rune Words | 透過 鐵匠鋪（Blacksmith） craft 產生 | 好 base + recipe + rune 投資後的終局製作品 |

品質階級則是另一條軸：

| Quality tier | 收集判斷 |
| ------ | ------ |
| `Normal` | 前期、特殊低等裝或高 sockets 工具裝；通常不要太早投入高階 rune |
| `Exceptional` | 中期成形與過渡，有些 set / unique 很值得收藏 |
| `Elite` | 後期長期目標，最需要比較 roll、base、slot、required level 與交易價值 |

## 多樣化取得途徑

不要只靠地城掉落。NGO 的收集路線更像多條管線一起跑：

| 來源 | 適合追什麼 | 注意 |
| ------ | ------ | ------ |
| 地城 / boss | 大量未鑑定裝、set、unique、legendary 候選、rune 與 base | 首通、Heroic、party size、con、任務掉落偏好都會影響期待 |
| Champion / Unique / Boss 怪 | 高稀有度或 magic+ 掉落期待 | 掉落數量與保證規則請以目前版本為準 |
| 酒館（Tavern）任務偏好 | 指定部位，例如 shield、plate、weapon 類 farming | 出發前先看任務目標與掉落偏好 |
| 商店巡視 | 過渡裝、caster base、plate / mail、socketed base | 藥劑店（Apothecary）、鐵匠鋪（Blacksmith）、商人（Merchant）偏好不同；restock 週期以 UI 為準 |
| Gambling | elite 飾品、Haniwa、Shako、特定 unique / set / legendary 候選 | 高風險 gold sink；先讀 [賭博（Gambling）指南](nevergrind-online-gambling.md) |
| Crafting | Mythical / Rune Words、職業鎖定 base、Superior / Ethereal 成品 | 先看 [屬性繼承規則](nevergrind-online-blacksmith-crafting-recipe-research.md#crafting-inheritance) |
| 共享銀行 | 分身裝、套裝件、抗性裝、交易品、craft base | 讓不同角色共用收集成果 |

<note>
<p>來源摘要常用「保證掉落」描述 Champion、Unique 或 Boss。不同 wiki 與版本對數量描述不完全一致；實務上可以把它理解成「較高價值掉落來源」，不要在公開筆記中把舊數量寫成永久規則。</p>
</note>

## 核心收集目標

### 詞綴與狀態

優先留意這些詞綴與狀態：

| 目標 | 為什麼重要 |
| ------ | ------ |
| `All Talents` / `All Skills` | 直接提高多數 build 的技能與天賦效率 |
| Talent tree / specific skill | 命中核心技能時，價值可能超過泛用詞綴；可用 [FC2 各職代表技能速查](nevergrind-online-fc2-signature-skills.md) 先判斷 |
| 高 resistance | 高難度地城與元素 boss 的生存門檻 |
| `Magic Find` / `Gold Find` | 提高 farming 期待，但要平衡 clear speed 與生存 |
| Sockets | 後期 rune plan 或工具裝價值來源 |
| `Superior` | craft base 的額外傷害或防禦價值 |
| `Ethereal` + `Indestructible` | 高基礎性能且抵消耐久風險，是很多玩家的終局收藏方向 |

### 裝備關鍵字

下面不是固定 BiS，而是來源摘要與既有筆記反覆提到、值得認名字的目標：

| 裝備 / 關鍵字 | 收集理由 | 延伸閱讀 |
| ------ | ------ | ------ |
| `Cryptic Paragon` / Haniwa | 施法職 / 支援 / 牧師（Cleric）常追的 `All Talents`、全抗、全法術傷與隨機技能武器 | [Haniwa 指南](nevergrind-online-cryptic-paragon-haniwa.md) |
| `Zamtil's Plenitude` | FC2 多個施法 / 支援職裝備例反覆採用的 elite 太腿部位；主要價值是補高難度地城所需的耐性與控場抗性，而不是只看 Enchanter 套裝歸屬 | [Set Items](nevergrind-online-set-items.md#zamtils-plenitude) |
| `Charlatan's Crest` / Shako | `All Talents`、屬性傷害與高 MF 類 farm 頭部目標 | [Unique Items](nevergrind-online-unique-items.md#unique-keywords) |
| `Demetrium's Ballista` | FC2 set 頁列為 `Socketed (1-6)`；若 roll 到高 sockets，即使低等也可能成為物理職 ranged slot 工具裝 | [Set Items](nevergrind-online-set-items.md#set-notable-pieces) |
| `Stormcaller Bow` | FC2 loot 頁把最高 socket roll 的 `Stormcaller Bow` 列為 Normal unique / set bow 裡值得帶回的例外；若剛好補 talents 或 rune space，先比較再賣 | [Unique Items](nevergrind-online-unique-items.md) |
| FC2 嚴選 Unique 速查 | FC2 `各職の厳選ユニーク` 頁整理了各職值得先留的 Unique / 特殊單件，包含 weapon breakpoint、class talent、skill enhancement、rare drop、主屬性與抗性用途 | [FC2 各職嚴選 Unique 裝備速查](nevergrind-online-fc2-selected-unique-items.md) |
| Elite amulet / ring / charm | gambling、跨職業與長期保留價值高 | [賭博（Gambling）指南](nevergrind-online-gambling.md) |
| Superior / Ethereal socketed base | Mythical / Rune Words 的核心素材 | [素材選擇優先級](nevergrind-online-blacksmith-crafting-recipe-research.md#crafting-base-selection) |

## FC2 loot 頁的帶回城判斷 {#fc2-loot-carry-priority}

FC2 的 [持ち帰るアイテム](https://atelier3.web.fc2.com/ngo/loot.html) 頁把「這趟要不要帶回城」拆成裝備收集與金錢效率兩件事。這是玩家整理的 meta snapshot，適合作為背包取捨規則；頁面本身也提醒 crafting 因為實裝較新，沒有納入該頁討論，所以 rune / crafting base 仍要回到 [符文](nevergrind-online-runes.md) 與 [製作與配方](nevergrind-online-blacksmith-crafting-recipe-research.md) 另外判斷。

| 候選 | FC2 頁面的判斷重點 | 本地筆記讀法 |
| ------ | ------ | ------ |
| Rare / Magic amulet | `All Talents +2` 很有價值；FC2 也提醒 magic amulet 可能出現這類詞綴。若同時有高 `Strength`，物理職可能用很久 | 飾品先看 talents、主屬性與抗性，不要因為不是 unique 就直接賣 |
| `Tree Talents +3` amulet | FC2 舉 `Judicator`、`Visionary`、`Elementalist` 作為可撐到 unique 前的 tree roll | 特定 build 還沒拿到固定 unique 時，rare amulet 可以先補技能樹缺口 |
| Rare gloves | 可能出現 `Tree Talents +2`；plate 職可在刷到目標 `Marshal Gauntlets Iniquity` 前過渡 | Talent roll 很稀有，也有替代品；先保留命中核心 tree 的手套，再慢慢比較 |
| Rare magic blunt / `Focus` / `Stave` | FC2 把 rare 魔法鈍器排在高價帶回優先級最前面，因為鑑定後 talents / caster mods 可能帶來自用或售價價值 | 背包快滿時，focus / stave 這類施法武器比很多低價小部位更值得鑑定 |
| Exceptional 以上 unique / set | FC2 建議從裝備收集與金錢效率兩面看，原則上都可以帶回 | 先帶回再整理；是否進銀行、交易或賣店，回城後看 roll 與部位決定 |
| Normal unique / set | 後期多數可以丟棄，但若會出 talents、元素抗性或是已知高 socket 例外，仍值得帶回 | 不要用「Normal = 一律垃圾」；已知工具裝或特定詞綴裝例外處理 |
| `Stormcaller Bow`、`Demetrium's Ballista` | FC2 特別點名最高 socket roll 可能留下來當終局工具裝 | 低等 bow 若 roll 到高 sockets，價值在 ranged slot 與 rune space，不在基礎傷害 |

## 收集與篩選流程

每趟地城後可以照這個流程清背包：

1. 先分出高稀有度、缺口部位、高價部位與 socketed base。
2. 到 [藥劑店（Apothecary）](nevergrind-online-apothecary.md) 使用 `Identify All` 類服務。
3. 先留自用候選：補抗性、資源、talents、core skill、haste 或 survivability。
4. 再留跨角色候選：set、unique、rare jewelry、抗性裝、high socket 工具裝。
5. 將分身、交易、套裝與 craft base 放入 [銀行（Bank）](nevergrind-online-edenburg.md)。
6. 其餘高售價裝備轉成 gold，支援技能、背包、補給、gambling 或 crafting。

<tip>
<p>銀行不是只放「現在穿不上但看起來強」的裝。更好的整理方式是用用途分類：分身裝、套裝件、抗性裝、MF 裝、gold value、craft base、待比較 unique。</p>
</tip>

## 什麼裝值得長期保留

| 問題 | 值得保留的訊號 |
| ------ | ------ |
| 它能補目前短板嗎 | 高抗性、高 HP、資源恢復、armor、命中或 threat / sustain 工具 |
| 它能改變 rotation 嗎 | `All Talents`、特定技能、cooldown、haste、bonus hit / proc 相關 |
| 它能服務多角色嗎 | 飾品、泛用 unique、set 件、高 resist 裝、MF / Gold Find 裝 |
| 它是好 base 嗎 | 正確 quality、socket count、Superior、Ethereal、職業詞綴或 skill enhancement |
| 它有交易價值嗎 | 常見高需求關鍵字、好 roll、好部位、稀有狀態或終局材料 |

如果一件裝備不能穿、不能賣好價、不能交易、不能給分身、不能當 craft base，也沒有特殊紀念價值，就不要讓它長期佔銀行格。
