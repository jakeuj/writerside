主要記錄下目前成功安裝的語法

`brew tap homebrew/cask-fonts`

`brew cask install font-sauce-code-pro-nerd-font`

# 先執行這行，才能用 homebrew 安裝字型。曾經執行過的人可以跳過這個指令 brew tap caskroom/fonts # 安裝指令 brew cask install font-sourcecodepro-nerd-font

`brew search nerd`
